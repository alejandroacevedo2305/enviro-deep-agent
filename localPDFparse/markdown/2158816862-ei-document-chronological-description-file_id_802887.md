# 7 Descripción de la fase de operación

## 7.1 Partes, obras y acciones asociadas a esta fase

49

La operación se inicia con el proceso de puesta en marcha del proyecto, en esta etapa se

consideran pruebas de funcionamiento para luego iniciar la operación propiamente tal. En

términos generales, la operación se realizará de forma remota, desde el Centro de Control

del titular. Junto con esto, se realizarán actividades de mantenimiento con la periodicidad

respectiva a cada tipo de mantenimiento a realizar, los cuales se detallan en el numeral 7.6.

## 7.2 Fecha estimada de inicio y término de la fase de operación.

función del tiempo estimado de la fase de construcción. El término de la fase de operación

será al cabo de 20 años, con lo cual corresponde a octubre de 2044 _._
