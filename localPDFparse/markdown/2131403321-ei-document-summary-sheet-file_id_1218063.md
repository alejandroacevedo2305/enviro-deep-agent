# Capítulo 9. Ficha Resumen

El artículo 19 letra f) del Reglamento del SEIA, D.S. N° 40/2012, señala que se deberá presentar una ficha en la
cual se resumen, para cada fase del proyecto o actividad, los contenidos a que se refieren las letras a), b), c) y d)
del presente artículo, a fin de facilitar la fiscalización a que alude el artículo 64 de la Ley. A continuación se
presenta la Ficha Resumen del Proyecto “Modificación Proyecto Técnico Piscicultura El Peral”.
