# Bins Isotérmico 1000 Litros con Tapa

Bins isotérmico con capacidad para 1000 Litros. Su color es gris estándar, además se
pueden fabricar con colores a pedidos.

Su estructura está diseñada para almacenamiento y transporte. Fabricado en polietileno
MAD con aditivos anti-UV, inyectado en poliuretano densidad 40kg/m3, isotérmico
doble pared, una sola pieza monoblock.
Cuenta con 4 entradas para grúa horquilla y transpaleta, tapa bipared apilable, y
desagüe opcional.

**Datos técnicos:**

- Capacidad: 1000 litros.

- Color: Estándar gris.

- Medidas interiores: Largo : 1095 mm. Ancho : 970 mm. Alto Fondo : 950 mm.
Dimensiones:

- Alto: 1210 mm con tapa.

- Largo: 1235 mm.

- Ancho: 1100 mm.

- Peso bins: 65 Kg.

- Tapa: 12 Kg.

Los datos técnicos entregados son aproximados y pueden variar según la configuración de los componentes
individuales. Superbidón se reserva el derecho de actualizar las fichas técnicas sin previo aviso.
