3. ANTECEDENTES GENERALES DEL PROYECTO

**3.1.** **Modificación de un Proyecto o Actividad**

De acuerdo con el Art. 12 del D.S. N°40/13 MMA, se señala que el Proyecto “Parque
Fotovoltaico Litoral Solar” corresponde a un proyecto nuevo y, por tanto, no se trata de una
modificación de un proyecto existente, por lo que no modifica Resoluciones de Calificación
Ambiental anteriores.

**3.2.** **Relación con las políticas, Planes y Programas de Desarrollo.**

**3.2.1.** **Estrategia Regional de Desarrollo Región de Valparaíso 2020**

La Estrategia Regional de Desarrollo (ERD) aprobada en el año 2012 es el instrumento que
permite avanzar al desarrollo y así definir la gestión del Gobierno Regional. Su imagen
objetivo es: “Una Región diversa, con un desarrollo Sostenible para el bienestar de sus
habitantes”. Junto con esta imagen objetivo. Se definen también los Ejes Estratégicos y sus
objetivos generales.
