El proyecto considera modificar el sistema de tratamiento de purines aprobado

bajo RCA N°385/2006, reemplazando el proceso aeróbico por un proceso de

digestión anaeróbica mediante la incorporación de biodigestores cerrados.

Por otra parte, como parte del proyecto se implementarán medidas para

asegurar un uso eficiente del agua en los pabellones, lo cual implicará una

disminución del volumen de purines generados. Finalmente, se modificará el

Plan de Riego vigente producto del cambio del sistema de tratamiento y de la

reducción de purines planteado.

La modificación sólo aplica a las actividades antes individualizadas,

manteniendo el resto de las partes aprobadas en la Resolución de Calificación

Ambiental vigente.

En el caso específico de la modificación del Sistema de Tratamiento de Purines,

la implementación de este tipo de sistema permite el manejo del purín dentro

de un sistema cerrado sin contacto con el ambiente externo.

En este contexto, cabe destacar que la experiencia en otros países,

principalmente de Europa, es concluyente, y se establece a este sistema de

tratamiento como el más adecuado para este tipo de efluentes. Lo anterior

sumado a que ya existe experiencia en Chile, específicamente en planteles de

cerdos tales como Plantel Las Pampas y Plantel Santa Irene, en la VI Región, en

donde ya existen biodigestores en plena operación hace más de seis año.
