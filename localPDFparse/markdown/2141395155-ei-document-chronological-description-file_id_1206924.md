**1. INTRODUCCIÓN**

Este capítulo presenta los cronogramas de actividades para las tres grandes fases del Proyecto:
construcción, operación y cierre. La fase de construcción considera una duración de 6 meses, la de
operación 30 años y para el cierre se prevé un tiempo de desmantelamiento de 4 meses. Cabe
destacar que los plazos aquí expuestos son referenciales y pueden sufrir modificaciones asociadas
a los contratos de compra de equipos y servicios.
