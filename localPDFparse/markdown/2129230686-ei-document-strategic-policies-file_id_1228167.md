**Relación con el proyecto**

Tal como se señala en la tabla anterior, el proyecto “Sistema de Tratamiento y
de Aplicación de RILes al Suelo” no se contrapone con las políticas y planes
evaluados ambientalmente, desarrollándose este en función de los objetivos
comunes de estos, velar por la integridad de los recursos y proyección del
medio ambiente, tanto para la construcción como operación del proyecto.
