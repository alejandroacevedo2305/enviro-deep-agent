El<br>cumplimiento de este compromiso se acreditará mediante el registro<br>fotográfico de la ejecución del compromiso. A partir de lo anterior, se<br>remitirá un informe redactado por profesional competente que dé cuenta<br>de todo el proceso y resultados alcanzados. |
