# Capítulo 7 Establecimiento de Inicio de Ejecución del Proyecto

Biocruz Generación S.A.

## Capítulo 7. Establecimiento de Inicio de Ejecución del Proyecto

La gestión [16] que dará cuenta del inicio de la ejecución del proyecto, considerando que se trata de la
conexión de un motor de combustión interna al interior de la Planta de Biocruz Generación S.A. para
la obtención de energía eléctrica, será la gestión para el envío de la Orden de Compra al contratista
para iniciar los trabajos de conexión del motor.

La fecha estimada de inicio de construcción será el 21/09/2016, esto sujeto a la fecha de obtención

de la RCA.

16 SEA ORD. No142034/2014. Define gestión como la realización de diligencias o tramites conducentes al logro de un
negocio, que en este caso correspondería a la ejecución del proyecto o actividad.
