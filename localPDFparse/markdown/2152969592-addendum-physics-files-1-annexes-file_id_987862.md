# ANEXO 3 CARACTERIZACIÓN ACCESOS PRIVADOS BY PASS 2X110 KV SAN RAFAEL

## 1. INTRODUCCIÓN

El Proyecto “By Pass 2x110 kV San Rafael” corresponde a una línea de transmisión de una longitud

aproximada de 7 km, para la cual se contemplan 22 estructuras reticuladas. Para el acceso a las

estructuras, se contempla la utilización de rutas públicas y caminos interiores existentes; no se

consideran actividades de habilitación de accesos, sino que se contempla la utilización de accesos

existentes en sus condiciones originales, sin ningún tipo de intervención.

## 2. OBJETIVO

 - Presentar registro fotográfico de los accesos de la red vial existente a utilizar para las

instalaciones de las estructuras eléctricas.

## 3. RESULTADOS

caminos interiores privados para las instalaciones de las estructuras de la línea
