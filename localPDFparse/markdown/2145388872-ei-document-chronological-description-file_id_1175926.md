# ANEXO 16

## Cronograma de Actividades

### CRONOGRAMA DE ACTIVIDADES

Como se observa en el cronograma del Proyecto (tabla 2), la etapa de Construcción se debe
realizar en un plazo máximo de 5 meses; el inicio del Proyecto está contemplado el año
2020 (mediados de septiembre) y la puesta en marcha 5 meses después de su inicio de
construcción. La etapa de Operación dura al menos 20 años pudiendo extenderse al cumplir
programas de mantenimiento preventivo y realizar actualizaciones de tecnologías.
