# 1 OBJETIVO

Este procedimiento establece la metodología para la detección y control de fugas en ductos de
transporte de gas.

# 2 ALCANCE

Este procedimiento es aplicable a Enap Magallanes.

# 3 TERMINOLOGÍA

Emergencia: cualquier evento fortuito o inesperado que se produzca en una o más instalaciones,
líneas de procesos, de transporte u otros procesos que ponga en riesgo la integridad de las
personas, el medio ambiente y las instalaciones, de causa diversa y gravedad variable que genera

la conciencia de una necesidad eminente de atención.

# 4 RESPONSABILIDADES

Los Jefes de Operaciones son responsables de difundir, aplicar, cumplir y hacer cumplir este

Procedimiento.

Los Operadores son responsables de cumplir este procedimiento.

# 5 DESCRIPCIÓN DE LA ACTIVIDAD

## 5.1 OPERACIÓN NORMAL

Durante la operación normal se considera la observación, registro y análisis de las variables de
proceso, la intervención de equipos y elementos de compresión, medición, registro y bloqueo
que aseguren el cumplimiento del objetivo para el cual el ducto fue instalado.

La medición registro y bloqueo puede ser realizada tanto en forma remota como local

## 5.2 OPERACIÓN EN EMERGENCIA

La fuga del gas transportado, ya sea por rotura violenta del ducto o pérdidas por escapes producto
de roturas de elementos de él, es considerada una emergencia.

Esta condición se detecta mediante la observación y análisis in situ o en forma remota de las
variables de proceso presión y flujo.

## 5.3 ACCIONES A SEGUIR

Descartado que los cambios observados en las variables correspondan a cambios en los procesos
aguas arriba o abajo del ducto, se procederá a:

- Aislar éste mediante las válvulas más cercanas, en forma remota si el sistema contempla
esta opción o en forma directa si no lo tuviera.

- Recorrer el trazado del ducto para ubicar el punto de fuga.

- Aislar el lugar donde se produce la fuga evitando riesgos de explosión o incendio.

- Aviso a la línea de supervisión

- Notificación a la autoridad competente si aplica de acuerdo a DS 280
