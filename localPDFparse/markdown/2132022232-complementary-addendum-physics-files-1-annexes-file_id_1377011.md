**SECCIÓN 10: ESTABILIDAD Y REACTIVIDAD**

**Estabilidad química** Estable bajo condiciones normales. Puede explosionar cuando es sometido a fuego o
golpes.
**Reacciones peligrosas** No polimeriza.
**Condiciones que se deben evitar** No almacenar cerca de fuentes de calor o chispa. No golpear.
**Materiales / sustancias incompatibles** Altamente reactivo con agentes reductores. Es reactivo con metales y álcali. Evitar
peróxidos, álcalis puede liberar humos de amoniaco.
**Productos de descomposición peligrosos** Óxidos de nitrógeno (NO x ), COx.
