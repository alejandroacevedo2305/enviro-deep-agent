# PRC-FL-0385 Procedimiento Operacional reabastecimiento de combustible Diesel.

Infraestructura de Datacenter - Facilites

10/03/2021

1

**1. Descripción**
Este procedimiento operativo tiene como objetivo instruir al analista para que lleve a cabo las operaciones
básicas dl proceso de descarga de combustible en el patio de generadores del DC.

**2.** **Objetivo**
Instruir a los colaboradores del área de Infraestructura para ejecutar de forma segura el reabastecimiento del
combustible de los generadores Diesel

**3. Aplicación:**
Este Procedimiento Operativo se aplica al área de Operación de DC - CHILE-2.
