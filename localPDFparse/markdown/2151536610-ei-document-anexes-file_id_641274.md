# Marzo 2021

El Proyecto Fotovoltaico Mantos del Sol se ubica en unas planicies aptas para el proyecto

fotovoltaico, que no presenta cursos de agua de importancia ni quebradas en su área. Sólo es
atravesada por dos pequeñas líneas de evacuación de aguas, las que serán respetadas.
