# Sección 1 _ SE-NW

Se representa el Piezómetro PMMH-03 junto a su planilla de detalle por sensor cuerda vibrante

Se representan los 2 pozos de bombeo más cercanos a dicho piezómetro, CN1P-03 y PBMH-09

Cada punto amarillo representa los sensores CV instalados en cada pozo, y la representación de la columna de agua
que registra a la fecha (línea negra discontinua).

El sensor ubicado en Grava Parda (sensor 4), es el que representa el Nivel Freático del acuífero inferior, y con el cual
se interpola el espesor saturado, en este caso representando un espesor de 12m.
