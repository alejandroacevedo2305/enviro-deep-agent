# 1.1 ARTÍCULO 16. ESTABLECIMIENTO DEL INICIO DE EJECUCIÓN DEL

PROYECTO.

En cumplimiento al artículo 16 del DS 40/12 se informa que la acción correspondiente al

movimiento de tierra corresponderá al inicio de la ejecución del Proyecto. Se estima que lo

anterior, será factible durante el segundo semestre del año 2016 o cuando se disponga de las

autorizaciones correspondientes.
