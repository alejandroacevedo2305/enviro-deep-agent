**1.** **DESCRIPCIÓN DEL PROYECTO DE MEDIA TENSIÓN**

La presente memoria descriptiva corresponde a la instalación de un transformador trifásico de
112,5 KVA. Este nuevo proyecto es de propiedad de Yura Chile y se construye para la
electrificación de la planta de revisión técnica. Las instalaciones señaladas se encuentran
ubicadas en sector industrial costado ruta A-65, Pozo Almonte, región de Tarapacá.

En este proyecto de construcción eléctrica, la realización de la obra se ejecutará según normas

actualizadas NCh.

El Nivel de Tensión de Líneas MT es de 23 KV propiedad de Eliqsa Cge S.A.

El Nivel de Tensión de Líneas BT es de 0,4 KV propiedad de Eliqsa Cge S.A..

**1.1** **PROPIETARIO**

El presente proyecto es propiedad de Yura Chile.

**1.2** **UBICACIÓN**

Las instalaciones a las que se hace referencia están ubicadas en sector industrial costado ruta A65, Pozo Almonte, Región de Tarapacá.

**1.3** **DESCRIPCIÓN ELÉCTRICA**

La presente memoria descriptiva corresponde a la instalación de un transformador trifásico de
112,5 KVA, además la instalación de un compacto de medida.

En S/E se instalará dos postes de hormigón armado MT de 11,5 metros de 525 Kg/Ruptura, un
transformador trifásico de 112,5 KVA, dos estructuras portantes de MT, dos mallas de tierra de 2
metros por 2 metros, un seccionador fusible de Baja Tensión de 3 x 175 [A] y tierras de servicio
tipo canastillo.
