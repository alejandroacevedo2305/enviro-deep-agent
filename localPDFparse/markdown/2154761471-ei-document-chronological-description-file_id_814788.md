**Cronograma de las Principales Partes, Obras y Acciones asociadas a la Fase de Cierre**

La fase de cierre considera una duración de 7 años, durante los 4 primero años se realizan las
actividades de cierre y luego por 3 años más las actividades de Post cierre. A continuación, en la se
presenta el cronograma con la secuencia de cierre de las instalaciones involucradas en el presente
Proyecto, en el cual se incluye el seguimiento y control como post-cierre.
