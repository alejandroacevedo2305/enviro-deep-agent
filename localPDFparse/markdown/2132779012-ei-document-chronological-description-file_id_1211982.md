# DESCRIPCIÓN CRONOLÓGICA DE SUS FASES

Respecto de los contenidos mínimos de las Declaraciones de Impacto Ambiental establecidos
en el DS 40/12 RSEIA, en este apartado se presenta el cronograma del Proyecto.

A continuación, se presenta el cronograma que detalla las actividades a realizar para cada
multipozo a perforar, el Proyecto tendrá una duración estimada de 20 años.
