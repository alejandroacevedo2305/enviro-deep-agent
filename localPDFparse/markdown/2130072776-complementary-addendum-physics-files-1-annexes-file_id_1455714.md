**Tabla 1: Ficha resumen antecedentes generales****

| Nombre del<br>Proyecto | Planta Solar Fotovoltaica Doña Antonia |
| --- | --- |
| Descripción breve<br>del Proyecto | El proyecto consiste en la instalación de 415.800 paneles fotovoltaicos para generar 100 MW<br>de potencia nominal instalada. La energía generada por la planta será transmitida, a través<br>de una línea subterránea interna hasta una Subestación Elevadora 23/110 kV para<br>posteriormente entregar la energía al Sistema Interconectado Central (SIC). |
| Objetivo del<br>Proyecto | Captar y transformar la energía solar en energía eléctrica para ser inyectada al Sistema<br>Interconectado Central (SIC). |
| Tipología principal<br>del Proyecto | D.S. N°40/2012 en el Artículo 3, literal c señala: Centrales generadoras de energía mayores<br>a 3 MW. |
| Tipología<br>secundaria del<br>proyecto | D.S. N°40/2012 en el Artículo 3, literal b señala: Líneas de transmisión eléctrica de alto<br>voltaje y sus subestaciones. |
| Modificación de<br>Proyectos<br>anteriores | El proyecto Planta Solar Fotovoltaica Doña Antonia corresponde a un proyecto nuevo, por lo<br>tanto no modifica proyectos anteriores. |
| Vida útil | La vida útil del proyecto se estima en 27 años, contando con una fase de construcción<br>estimada en 11 meses, una fase de operaciones de 25 años y una fase de cierre de 11<br>meses. |
| Gestión, acto o<br>faena mínima que<br>da cuenta del<br>inicio de la<br>ejecución | El acto que dará cuenta del inicio de ejecución del proyecto será la Instalación del primer<br>contenedor dentro del área de instalación de faenas. Mientras que el acto que dará por<br>realizada la fase de construcción será la conexión de la subestación eléctrica al SIC. |
| Desarrollo de<br>proyectos o<br>actividades por<br>etapas. | El proyecto no se realizará por etapas. |
