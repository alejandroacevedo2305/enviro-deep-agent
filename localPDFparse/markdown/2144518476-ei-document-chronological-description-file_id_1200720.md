**2.5.4. Mano de obra fase de construcción (Art. 19 a.5.4.)**

El promedio de mano de obra durante la fase de construcción corresponderá a 327
personas, mientras que el momento de mayor requerimiento requerirá de las labores
realizadas por un máximo de 740 personas.

El horario de trabajo será de lunes a viernes de 9:00 a 18:00 horas y los días a sábado de
9:00 a 13:00 horas.

**2.5.5. Suministros básicos fase de construcción (Art. 19o a.5.5.)**

Los insumos que se utilizarán para la fase de construcción serán provistos según el detalle
que se presenta a continuación.
