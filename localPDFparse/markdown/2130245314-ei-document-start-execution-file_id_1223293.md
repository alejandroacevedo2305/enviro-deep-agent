**1** **ESTABLECIMIENTO DEL INICIO DE EJECUCIÓN DEL PROYECTO**

El acto o faena mínima que dará inicio a la ejecución del proyecto consistirá en la instalación de faenas en el
área de emplazamiento del proyecto.
