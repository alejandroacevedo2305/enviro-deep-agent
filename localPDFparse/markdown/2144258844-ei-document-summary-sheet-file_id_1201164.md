# 9. Fichas Resumen

Las fichas resumen se presentan en el Anexo 21, las cuales incluyen la Ficha descripción de

proyecto; la Ficha de antecedentes necesarios que justifiquen la inexistencia de aquellos

efectos, características o circunstancias del artículo 11; la Ficha plan de cumplimiento de la

legislación ambiental aplicable; y la Ficha compromisos ambientales voluntarios.
