**2.2.5** **Vida útil y cronograma de etapas.**

El proyecto Sistema de Tratamiento para disponer RILes al suelo, ha sido
evaluado económicamente con un horizonte de 30 años de operación, no
definiéndose un cierre propiamente tal, sino el mejoramiento continuo de las
instalaciones, adoptando mejoras tecnológicas en ellas.
