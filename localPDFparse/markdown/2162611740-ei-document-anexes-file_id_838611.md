# Comacchio MC800 #2005

Perforadora hidráulica marca Comacchio de origen Italiano, MC 800, montada sobre
carro de orugas. La unidad de rotación que posee permite perforar con una amplia variedad de
sistemas de perforación, gracias al alto torque que puede desarrollar (16.800 Nm) con
velocidades de hasta 86 rpm.
