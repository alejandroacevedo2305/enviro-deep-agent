**PRODUCCIÓN DE BASURAS DEL EDIFICIO**

Total Departamentos = 150
Total personas = 410
Factor Basuras Kg / hab*dia = 0,995 Kg
410 Personas x 0,995 Kg / hab*dia = 407,95 kg/hab*dia
407,95 kg x 3 días = 1.224 Kg
1.224 kg / 150 Kg/m3 = 8.160 lt

Total contenedores de 240 litros: 34

Total Parrillas = 6 unidades de 1.30 mts
