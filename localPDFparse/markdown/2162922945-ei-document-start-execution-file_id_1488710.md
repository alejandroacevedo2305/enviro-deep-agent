**1.4.9** **Plazo e hitos estimados de inicio del Proyecto**

La ejecución de las obras y actividades del Proyecto darán inicio una vez se obtenga la RCA favorable y las

respectivas autorizaciones sectoriales de funcionamiento. En la **Tabla 1-3** se presentan los hitos que dan

inicio y término a cada una de las fases.
