de Lixiviación - Spence”.

Además, el presente Proyecto contempla las siguientes medidas de control de emisiones para las fases de
construcción y operación:

Fase de Construcción

 - Velocidad de circulación para camiones de 40 km/h y 50 km/h para buses y camionetas.

 - Humectación de caminos y frentes de trabajo.

Fase de Operación

 - Estabilización de caminos mediante la humectación con una solución de bischofita o producto
equivalente alcanzado un 80% de eficiencia.

 - Humectación de caminos, al menos 3 pasadas diarias de camiones aljibe. Esta frecuencia fue
estimada con el objetivo de mantener la eficiencia de control de emisiones en 40%.

 - Humectación diaria de los frentes de trabajo, antes de realizar las actividades.

 - Velocidad de circulación para camiones de 40 km/h y 50 km/h para buses y camionetas

(incluido el factor de corrección por incertidumbre), inferior al aporte máximo de 18,4 ug/m [3] para el caso base

asumiendo que todas las otras condiciones que influyen en el proceso de generación de caídas de roca (por
