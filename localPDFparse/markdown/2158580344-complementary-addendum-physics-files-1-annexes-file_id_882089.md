**2.0** **DESCRITIVO DO ESCOPO DE FORNECIMENTO**

Nesta revisão A da proposta estamos ofertando o sistema de tratamento de gases do forno,
considerando a mesma especificação técnica do forno projeto Verallia Jacutinga - Forno 2.

Estamos ofertando o sistema em duas opções:

**OPÇÃO A:**

A. SISTEMA COMPLETO: Semi-Dry FGD (com QR), ESP, DeNOx (Sistema de Aquamônia,
SMX e SCR), Sistema de exaustão do HEC, 2 IDF ́s, Dutos, Dampers, suportes. Juntas de
expansão e Chaminé.

**OPÇÃO B:**

B. SISTEMA SEM DeNOx: Idem acima, sem o Sistema DeNOx.

**As eficiências consideradas são:**

a) DeSOx (Sistema Semi-Dry FGD) com eficiência > 95%;

b) ESP para garantia de emissão < 10 mg/Nm3, base seca, com correção 8% O2;

c) Sistema DeNOx com Eficiência > 50%.

**Folha de dados do ESP - Anexo:**
