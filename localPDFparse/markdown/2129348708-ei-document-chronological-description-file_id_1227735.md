# DESCRIPCIÓN CRONOLÓGICA DE SUS FASES

Una vez que la Resolución de Calificación Ambiental haya sido emitida (mes 0), se procederá
a la Etapa de Construcción, para la cual se estiman 2 meses de ejecución para cada ducto,
mientras que para la Etapa de Operación de las líneas de flujo, se estima una duración de 20
años. El programa de actividades se detalla en el siguiente cronograma equivalente a cada

ducto:
