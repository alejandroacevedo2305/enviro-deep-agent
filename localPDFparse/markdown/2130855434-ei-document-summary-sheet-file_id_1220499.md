|Objetivo|Por medio del proyecto que se somete a evaluación ambiental, se pretende<br>modificar algunos aspectos de la Planta Pesquera Álvarez y Álvarez Ltda., de tal<br>forma de ampliar la capacidad diaria de proceso de Materia Prima (MMPP),<br>para la elaboración de productos terminados en base a salmónidos, con un<br>aumento sostenido en la capacidad de proceso de Materia Prima desde 13<br>Ton/día a 42 Ton/día1 el segundo semestre de 2016.<br> <br>Para ello, se construirán, instalarán y/o se optimizarán algunos componentes de<br>la planta de proceso o asociados a ella, necesarios para garantizar un proceso<br>productivo eficiente que cumpla con la normativa ambiental vigente, lo cual<br>será un condicionante para cada una de las etapas de la presente modificación.<br>|
