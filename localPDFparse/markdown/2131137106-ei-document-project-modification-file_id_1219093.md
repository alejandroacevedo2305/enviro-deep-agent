_**1.3.8**_ _**M**_ _**ODIFICACIÓN A PROYECTO**_

El proyecto contempla intervenir o complementar mediante la estimulación del
reservorio, la fase de perforación de los pozos Konawentru 17 y Pampa Larga 16,
las cuales están calificadas bajo la RCA N° 98/2012.
