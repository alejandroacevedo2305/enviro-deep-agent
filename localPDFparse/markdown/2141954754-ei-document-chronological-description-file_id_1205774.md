**1.3.8.** **Levantamiento de Información**

La etapa de levantamiento de información, comienza con el recorrido por sector, en

donde se pretende extraer el árido, por lo cual se consideran factores como la calidad del

material y su cercanía al área en donde se ejecuta el proyecto., los especialistas

desarrollan sus especialidades en los sectores aledaños, mediante la realización de

calicatas se define el área a explotar, luego Topografía ejecuta un levantamiento con el

objetivo de definir los volúmenes disponibles de material y georreferenciar las áreas de

intervención del proyecto.

Todos los estudios, análisis e informes se realizan con la finalidad de determinar las

características del sector que se intervendrá, su área de influencia, identificar si existen

áreas aledañas sensibles que puedan verse afectadas, además de establecer acciones o

medidas correctas acorde a la aplicación de la normativa ambiental vigente aplicable a

este tipo de proyectos.

**1.5.6.** **Descripción de cómo se proveerá durante esta fase de los suministros**

**básicos tales como energía, agua, servicios higiénicos, alimentación,**

**alojamiento, transporte y otro semejante.**

El proyecto contempla la generación eléctrica para las cintas transportadoras y la

chancadora. Las cintas trasportadoras Kleemann z16 son una unidad, es decir cada cinta

tiene incorporada internamente un generador propio, el cual funciona con petróleo, el

a través de camión tanque externo, el cual tendrá toda la documentación vigente para

realizar este tipo de servicios.
