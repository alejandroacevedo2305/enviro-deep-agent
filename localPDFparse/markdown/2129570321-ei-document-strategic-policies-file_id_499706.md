# Relación con las políticas y planes evaluados estratégicamente

De la revisión de catastro público de la Evaluación Ambiental Estratégica de la plataforma
web del Ministerio de Medio Ambiente ( [http://eae.mma.gob.cl/index.php/ficha ) se](http://eae.mma.gob.cl/index.php/ficha)
constata que entre no la comuna de Santa María y la región de Valparaíso no existen
políticas ni planes evaluados estratégicamente a la fecha.
