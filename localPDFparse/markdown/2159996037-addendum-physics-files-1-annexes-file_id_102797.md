1. Introducción

El presente informe detalla la salida a terreno realizada con el objetivo de inspeccionar y verificar la

presencia de cauces en el área de influencia de la piscicultura ubicada en Ensenada Ruta U-99-V. La

tarea principal consistía en realizar un plan de vuelo con el dron para generar un orto mosaico, que

permitieran identificar posibles cauces adicionales en la zona. Sin embargo, debido a condiciones

ambientales adversas, se debió recurrir a la toma de imágenes manuales para cumplir con los

objetivos planteados.

2. Sistema de referencia

El sistema de referencia utilizado corresponde a Datum WGS 84, Proyección Cartográfica UTM Huso

18S (Universal Transversa de Mercator), para la configuración de GPS navegador Garmin Oregón

750, y Dron que se utilizó fue un DJI Mavic 3 Enterprise.

3. Proceso de levantamiento

El procedimiento utilizado se dividió en diferentes etapas requeridas para la obtención de las

exactitudes finales del proyecto. Se enuncian a continuación:

3.1. Vinculación de PRS

En terreno se realizó una búsqueda exhaustiva para identificar los puntos de control (PR),

identificando dos, los cuales son clave para asegurar la precisión de nuestro trabajo. Para verificar

la concordancia de las coordenadas con el plano de referencia, se utilizó un GPS manual. Mediante

este equipo, se georreferenciaron dos puntos situados fuera del perímetro de la piscicultura,

específicamente en la Ruta U-99-V.

Con el objetivo de facilitar la identificación de estos puntos durante el vuelo fotogramétrico, se

procedió a colocar dianas sobre los PRS y se georreferenciaron. Estas dianas servirán como

referencias visuales y contribuirán a la exactitud del proceso de captura de imágenes, garantizando

que los datos recogidos sean precisos y fiables para el análisis posterior.

3.2. Planificación del Vuelo con Dron

Se intentó generar el plan de vuelo con el dron en una zona abierta, la playa, que proporcionaba

una buena señal satelital, sin embargo, la altura de los árboles impidió que el dron superara la copa

de estos, alcanzando la altura máxima autorizada por la DGAC.

**Figura 3.** Plan de vuelo con dron

3.3. Intento de vuelo dentro de Piscicultura

Se realizó un segundo intento de planificar el vuelo dentro de la piscicultura. No obstante, está

ubicada en una quebrada, lo que generó problemas con la señal satelital del dron, impidiendo

ejecutar el plan de vuelo.

3.4. Vuelo Manual

Debido a las dificultades anteriores, se procedió a tomar imágenes de forma manual con el dron

para poder visualizar si había más cauces de los que ya están identificados, por lo que también se

recorrió la zona de interés caminando, tanto dentro de la piscicultura como por la carretera

circundante.

3.5. Trayectoria de vuelo

Se puede apreciar la trayectoria de vuelo que realizó el dron, lo cual resulta fundamental para

comprender en detalle los cauces presentes en la zona de influencia. A través de la visualización de

este recorrido aéreo, se facilita el análisis y la identificación de los diferentes cursos de agua y sus

características.

**Figura 4.** Trayectoria de vuelo primera visita a terreno

Se realizó una segunda visita al terreno, durante la cual se ejecutó el plan de vuelo que cubría el

área de influencia del KMZ del cliente. Este archivo KMZ se exportó al control del dron para llevar a

cabo el vuelo. Con las imágenes capturadas, se generó un ortomosaico.
