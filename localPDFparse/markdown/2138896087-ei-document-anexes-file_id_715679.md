## 1. INTRODUCCION

Esta memoria de cálculo está desarrollada para determinar los diámetros
de las redes de AGAPACK que alimentarán diferentes puntos de consumo
distribuidos al interior de la planta de Producción de Tottus ubicada en la
comuna de Quilicura en la ciudad de Santiago.
El proyecto contempla el suministro de 3 distintas mezclas de Gases para
uso alimenticio, las que se indican a continuación:

 AGAPACK 2.2

 AGAPACK 3.1

 AGAPACK 4.0

## 2. OBJETIVOS

 Determinar perdidas de carga
 Determinar los diámetros de diseño de cada red

consumos indicados anteriormente, aplicando un factor de simultaneidad
de 100%. A su vez se consideró una presión de salida de cada central de
suministro de 8 [bar].
El estudio se realizó dividiendo en tramos el tendido de la red y para estimar
las pérdidas por singularidades se estimó un largo equivalente para cada

tramo en estudio.

mínimos por lo que no serán considerados debido a que no influirán
