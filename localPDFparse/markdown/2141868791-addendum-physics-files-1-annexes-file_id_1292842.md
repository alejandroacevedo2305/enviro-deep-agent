**Respuesta:**

El estanque a utilizar para el almacenamiento temporal de las aguas servidas tendrá una
capacidad para almacenar 20.000 litros, y tendrá las siguientes dimensiones:

Altura: 29 cms. Aprox.

Diámetro Superficie: 321 cms. Aprox.

Diámetro Inferior: 321 cms. Aprox.

El estanque será instalado sobre una base nivelada, sin piedras, ni imperfecciones, ya que
debe soportar el peso total del estanque (incluido el líquido). La instalación puede ser
subterránea o superficial, de modo que dependerá de las condiciones del terreno el tipo de
intalación que se tendrá.
A modo general, las prevenciones que se tomarán para evitar la infiltración de aguas
servidas al suelo será mantener una mantención periódica, según indicaciones del
proveedor, donde se realizará una inspección visual del estado del estanque. Asimismo, se
realizará un vaciado periodico, de modo que la planta no alcance la máxima capacidad de
almacenamiento.

El el caso particular que el estanque se instale bajo la superficie del suelo, se propone como
medida no rellenar los costados con tierra de modo de no generar presión en las paredes
del estanque, y construir una losa de hormigón en la parte superior e inferior del estanque.
Se reitera que dicha medida se realizará sólo en el caso de instalar el estanque bajo tierra.
