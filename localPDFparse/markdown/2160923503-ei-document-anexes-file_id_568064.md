# 7 COMPROMISO DE SOMETERSE A EVALUACIÓN Y CERTIFICACIÓN DE RCA

Resolución de Calificación Ambiental. En consecuencia, no procede el pronunciamiento sobre esta materia

en relación con el presente Proyecto.
