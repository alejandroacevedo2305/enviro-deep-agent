**SECCIÓN 4: Primeros Auxilios**

**En caso de contacto accidental con el producto, proceder de acuerdo con.**

**Inhalación** : No irrita el tracto respiratorio.
**Contacto con la piel** : No produce irritación.
**Contacto con los ojos** : El contacto con los ojos puede causar
irritación, quemazón y lagrimeo. Enjuagar
con abundante agua durante 15 minutos.

**Ingestión** : Puede causar náuseas y vómitos. No Inducir al
vómito, en el caso de que la persona este
inconsciente acudir a un médico.

**Notas para médico tratante** : Si la persona esta inconsciente y tiene problemas
de respiración, no inducir al vomito ni dar agua.
