# 1.5 Modificación de Proyecto

modificación de un proyecto existente.

De acuerdo a lo solicitado en el Artículo 14 del DS N°40/12 MMA, el Proyecto **se realizará en una**

del proyecto, de modo sistemático y permanente es el siguiente:
