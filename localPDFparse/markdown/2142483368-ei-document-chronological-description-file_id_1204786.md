**Fechas Fase de Cierre**

El comienzo de la fase de cierre se tiene programada para el 02 de mayo del año 2048, y
estará dada por el cese de funcionamiento de las instalaciones. El término de esta fase
estará dado por el retiro total de las instalaciones, estimándose para el 26 de diciembre del
año 2048, considerando 6 meses para esta fase
