**4.- MEDIDAS DE PRIMEROS AUXILIOS**

En el caso de contacto accidental con el producto, proceder de acuerdo con:

|Inhalación|Apartar de exposiciones posteriores. Si hubiera irritación<br>respiratoria, vértigo náusea o inconsciencia, solicitar ayuda<br>médica inmediatamente. Si hubiera parada respiratoria aplicar<br>máscara de ventilación forzada o respiración boca a boca.|
|---|---|
