#### 13 Cronograma de las fases del proyecto

De acuerdo con los requerimientos de la letra a.5 y a.6 del artículo 19 del D.S. N°40/2012 MMA, la DIA

debe incluir un “cronograma de las principales partes, obras y acciones asociadas a esta fase (construcción

y operación), utilizando cualquier herramienta de representación gráfica del progreso del proyecto o

actividad”.

Estimativamente, el proyecto considera una vida útil de 55 años, que considera todas las fases del proyecto.
