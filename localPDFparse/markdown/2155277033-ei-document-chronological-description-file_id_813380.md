**1.5.4** **Mano de obra requerida durante la fase de construcción**

Para la ejecución de los trabajos de la fase de construcción, se estima un promedio

de 90 y máximo de 100 trabajadores. Esta mano de obra es considerada para todas

las actividades requeridas en la fase de construcción (obra gruesa, terminaciones,

urbanizaciones, obras exteriores, instaladores, retiro de escombros y materiales de

desechos, etc.).
