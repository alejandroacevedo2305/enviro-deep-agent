**1.2.8.** **Inicio de la Ejecución del Proyecto**

Se considera como inicio del proyecto el inicio de la fase de construcción de las instalaciones

anexas a las ya existentes, la que se inicia 5 días hábiles una vez obtenidos los permisos

ambientales.
