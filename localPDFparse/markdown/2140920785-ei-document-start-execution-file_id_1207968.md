## Capítulo 7. Establecimiento de Inicio de Ejecución del Proyecto

corresponde al cambio de redes de servicio. Se establece como probable fecha de inicio el
01/04/2019.

La fecha indicada está sujeta a la fecha de obtención de la RCA.
