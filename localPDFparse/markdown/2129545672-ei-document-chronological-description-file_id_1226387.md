CRONOGRAMA DE ACTIVIDADES

Una vez que la Resolución de Calificación Ambiental ha sido emitida (mes 0), se dará inicio a
la Etapa de Construcción de cada línea de flujo, para lo cual se estima 1 mes de ejecución por
cada ducto proyectado.

La Etapa de Operación de este Proyecto contempla una duración de 20 años, sin embargo y
debido a mantenciones programadas, tales como cambios de tramos por desgaste de
material, es posible extender su vida útil. El programa de actividades se detalla en el
siguiente cronograma.
