# ANEXO J INFORMACION CLIMÁTICA MEJILLONES

1

Se presenta en este anexo mayores antecedentes climáticos de Mejillones, donde se encuentra
el sitio de almacenamiento temporal de neumáticos usados.

**1.** **Tipo de Clima**

El clima en Mejillones es del tipo "desierto". No hay virtualmente ninguna lluvia durante el
año. El clima en esa localidad se clasifica como BWh por el sistema Köppen-Geiger.

**2.** **Pluviometría**

En Mejillones, el mes más seco es enero, con 0mm de lluvia. La mayor parte de la
precipitación cae en julio, promediando 1mm.

**3.** **Temperatura**

La temperatura media anual es de 18.3 °C. A continuación, se presenta un diagrama de
temperatura durante el año.
