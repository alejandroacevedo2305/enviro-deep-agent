# SIERRA CIRCULAR DE 7.1/4 ́ ́1400W 5500RPM

**Código:** 851.3361

**Marca:** DEWALT

**Descripción**

Sierra circular de 7. 1/4", modelo DWE560-B2C, marca

DEWALT.

Compacta y ligera con 65mm de profundidad de corte,
motor de alta potencia de 1.350W, que proporciona un
mayor rendimiento de corte, diseño de la guarda interna
cortada para una mejor visibilidad de línea de corte para
máxima precisión. El soplador de polvo limpia el polvo y
la suciedad de la línea de corte, eficaz salida del polvo
para minimizar las partículas de polvo en el aire cuando
se conecta a un aspirador. Compatible con el sistema
airlock para una conexión sencilla y segura de la
manguera del aspirador.
