# Modificaciones de un proyecto o actividad

En conformidad a lo indicado en el Artículo 12 del D.S. N° 40/2012 MMA, el Titular señala

que el proyecto que se somete actualmente a evaluación ambiental viene a modificar a un
proyecto que ya fue evaluado ambientalmente por la Autoridad.

El proyecto original, fue aprobado mediante RCA N°128/2011, en la siguiente Tabla se
resumen las modificaciones que considera el presente Proyecto.
