## 1. INTRODUCCIÓN.

El presente Anexo contiene el detalle del cálculo de la estimación de emisiones

atmosféricas, para las fases de construcción y operación del proyecto **“Modificación Uso**

**de Bodega de Sustancias Peligrosas”** el cual se ubicará en la comuna de Lampa, Región

Metropolitana.

Para la estimación de emisiones se consideró la velocidad del viento del sector. La fuente

de este dato corresponde a un informe de monitoreo meteorológico correspondiente a la

Estación Quilicura I perteneciente al Sistema de Información Nacional de Calidad del Aire.

En ese informe, se desprende que el 99% del tiempo el viento está por debajo de los 5,4

m/s (ver imagen de datos de 2015 que se anexa a presente memoria).
