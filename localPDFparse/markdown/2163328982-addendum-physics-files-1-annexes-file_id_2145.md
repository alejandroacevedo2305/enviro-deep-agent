El cálculo de la demanda de energía (Q) se calcula a partir del coeficiente volumétrico global de pérdidas térmicas totales
(G V2 ), que a su vez depende del coeficiente volumétrico de pérdidas térmicas por transmisión de envolvente (G V1 ) .
