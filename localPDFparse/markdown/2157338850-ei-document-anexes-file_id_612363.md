**Capacidad del canal Los Cantos sin proyecto**

La capacidad original del canal está dada por la sección y pendiente de la alcantarilla

por la que cruza el canal bajo la Avenida Salvador Allende.

Dicha alcantarilla corresponde a un tubo de cemento comprimido de 1.200 mm de

diámetro.

Para determinar la capacidad hidráulica del tubo de cemento comprimido de 1.200 mm

de diámetro existente se utilizará la fórmula de Manning, expresada de la siguiente

manera:

Q = (1/n) x (R [2/3] ) x (i [1⁄2] ) x (A)

Donde:

_Q : Máxima capacidad hidráulica de las cunetas_
_A : Área sección (m_ _[2]_ _)_
_i_ _: Pendiente longitudinal (m/m)_
_R : Radio hidráulico (m)_

_n : Coeficiente de rugosidad de Manning._

Q = (1 / 0,015) x (0,365 [2/3] ) x (0,01 [1⁄2] ) x (0,97)

una capacidad máxima de 3.481,8 [l/s], el cual es el caudal máximo que fluye en canal

Los Cantos aguas debajo de este cruce.
