- Volteo del alperujo

_a.2._ _Descripción de variables meteorológicas relevantes_

El sector se caracteriza por tener un clima mediterráneo marino, con temperatura media

territorio.

El área de localización de la planta corresponde a un clima templado cálido con lluvias
