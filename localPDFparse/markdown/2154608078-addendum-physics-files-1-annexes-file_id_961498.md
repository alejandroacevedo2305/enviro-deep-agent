|**Métodos de limpieza**|Siempre que existan derrames o fugas de Hipoclorito de<br>Sodio se deben efectuar los siguientes pasos:<br>Ventilar el área de fuga o derrame<br>Recolectar el material derramado en la manera más<br>conveniente para su reciclaje o para su disposición en<br>un relleno de seguridad.<br>Si el derrame es pequeño, el Hipoclorito de Sodio se<br>puede absorber en vermiculita, arena seca o un<br>material similar, que no sea combustible, luego de lo<br>cual se puede neutralizar con soluciones de Sulfuro de<br>Sodio (Na2S), Sulfito de Sodio (Na2SO3), o Tiosulfato<br>de Sodio (Na2S2O3).<br>Si el derrame es grande y en un área cerrada, se puede<br>diluir con agua y luego se puede neutralizar con alguna<br>de las soluciones mencionadas.|
