**Pauta de observación etnográfica:**

Criterios que deben ser observados en campaña de terreno en virtud de los contenidos mínimos
que deben tener DIA y EIA según D.S. 40/12 del RSEIA.

**Dimensión Geográfica** :

- Líneas de transporte público

- Paraderos de locomoción colectiva

- Avenidas principales

- Calles más transitadas

**Dimensión demográfica** :

- Sedes de adulto mayor

- Clubes deportivos

- Plazas con juegos infantiles

**Dimensión de bienestar social básico** :

Salud:

- Pública: CESFAM, CECOSF, Hospital

- Privada: Consulta médica, Clínica.

Educación:

- Colegios: Ubicación, nivel de enseñanza

- Universidades

**Dimensión Antropológica** :

- Elementos patrimoniales

- Sitios de significación cultural

- Lugares de reunión comunitaria

- Sedes sociales

- Canchas deportivas

**Dimensión Socioeconómica** :

- Negocios locales

- Ferias libres

- Supermercados
