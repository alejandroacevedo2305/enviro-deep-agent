## DESCRIPCIÓN CRONOLÓGICA DE SUS FASES

La ejecución del Proyecto tendrá una duración aproximada de 25 días por pozo, cuyo inicio se
estima para abril del año 2018, iniciando con las actividades de montaje de equipos para
proceder a la fracturación del pozo. Como se mencionó en la DIA, la fracturación de los pozos
se realizará de forma consecutiva y no simultánea.

A continuación, se presenta el cronograma que detalla las actividades a realizar en cada pozo

a fracturar.
