# ANEXO 2.4 Sistema de Potabilización del Agua del Equipo

**1. CARACTERISTICAS DEL EQUIPO CLORADOR**

El sistema de cloración de agua en línea, consiste en la disolución de
tabletas de cloro, a través del flujo del agua a potabilizar a la salida del
estanque de almacenamiento.

Consiste en una cámara donde se almacenan las tabletas de cloro (hasta
1,9 kilos).

El sistema no requiere de energía eléctrica para su funcionamiento

**2. VENTAJAS SISTEMA**

 Facilidad de uso

 Tapa de fácil abertura, permite revisar el estado de las tabletas y
cerrar con seguridad

 Permite dosificar el cloro en la red

 Permite mantener la ppm de cloro libre entre 0 - 2.0, de acuerdo
a lo requerido por la normativa.

 No requiere energía eléctrica para funcionar.

 Facilidad de instalación.
