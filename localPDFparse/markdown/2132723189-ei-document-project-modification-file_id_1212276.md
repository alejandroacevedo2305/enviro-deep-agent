_**“ESTIMULACIÓN**_ _**HIDRÁULICA**_ _**POZO**_ _**UAKEN**_ _**X-1” 1.3.8 MODIFICACIÓN A PROYECTO**_

_**1.3.8**_ _**M**_ _**ODIFICACIÓN A PROYECTO**_

El proyecto contempla intervenir o complementar mediante la estimulación del
reservorio, en caso de ser necesario, la fase de perforación del pozo Uaken x-1, la
cual está calificada bajo la RCA N°98/2012.
