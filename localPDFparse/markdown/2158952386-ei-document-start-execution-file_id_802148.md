Capítulo 1 - Descripción de Proyecto
DIA “Aumento capacidad de almacenamiento de combustible Distrito Centinela”

# 1.4.9 Plazo e hitos estimados de inicio del Proyecto

El Proyecto se iniciará una vez aprobado ambientalmente y luego de obtener las autorizaciones sectoriales

pertinentes. Los hitos previstos para dar cuenta del inicio y fin de cada fase de presentan a continuación.

**Tabla 1-4. Hitos de inicio y fin de cada fase del Proyecto**

|Fase|Inicio|Col3|Fin|Col5|
|---|---|---|---|---|
|**Fase**|**Hito**|**Fecha Estimada**|**Hito**|**Fecha Estimada**|
|Construcción|Habilitación<br>instalación<br>de<br>faena de estación fija.|Noviembre 2023|Retiro instalación de faena de<br>estación fija.|Noviembre 2024|
|Operación|Abastecimiento<br>de<br>combustible estación fija.|Noviembre 2024|Cese de abastecimiento de<br>combustible desde todas las<br>estaciones proyectadas.|Marzo 2064|
|Cierre|Inicio<br>de<br>actividades<br>de<br>desmantelamiento|Abril 2064(1)|Término de actividades de<br>desmantelamiento|Marzo 2066|

(1) Como se indicó previamente, el desmantelamiento de las instalaciones se realizará de manera paulatina y

conforme se avance con las labores de cierre de las demás instalaciones del distrito.

Fuente: GAC

Las fechas previamente indicadas se condicen con aquellas previstas para el proyecto Desarrollo Minera

Centinela (DMC) de acuerdo con la evaluación ambiental más reciente (RCA N°267/2021). De esta manera,

las obras y actividades objeto de la presente evaluación no modifican ni extienden la vida útil de la faena

de Minera Centinela.

Sin perjuicio de lo indicado, la fecha precisa de cada hito será informada a la SMA.

# 1.4.10 Desarrollo de Proyecto o actividad por etapas

El Proyecto que se somete a evaluación no contempla su desarrollo por etapas.
