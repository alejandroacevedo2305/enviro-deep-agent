Descripción Cronológica de las Fases del Proyecto

La duración de la Etapa de Construcción del Proyecto será de aproximadamente 6 meses,
y contempla las actividades que se identifican en el cronograma que se presenta en la Tabla
1.

Se dará inicio a la etapa constructiva una vez obtenida la RCA, estimando el inicio de las
obras en octubre de 2018, con la habilitación de la instalación de faenas. Se dará término a
esta etapa con la realización de las tareas relativas al desmantelamiento de las

instalaciones de faenas.

Para la Fase de Operación del Proyecto se contempla de una duración máxima de 20 años
en cumplimiento con el artículo 55 del D.S. N° 189/2008. Se entiende que se dará inicio a
la Fase de Operación con el comienzo de las mantenciones y monitoreos ambientales, una
vez terminada la fase de construcción de las obras, y se dará término a esta etapa con la
realización de los últimos monitoreos ambientales correspondientes.
