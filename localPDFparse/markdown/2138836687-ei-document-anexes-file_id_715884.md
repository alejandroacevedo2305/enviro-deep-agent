## INTRODUCCIÓN

Como parte de los estudios y antecedentes requeridos para la caracterización de los suelos
se ejecutó la exploración geotécnica mediante calicatas para determinar la estratigrafía del

sector.

Se realizaron 6 calicatas en el área de ejecución del Proyecto con el fin de recopilar mayores
antecedentes, los cuales se presentan a continuación.
