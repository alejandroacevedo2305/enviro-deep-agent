# Fecha estimada de inicio y término de la fase de cierre o abandono.

Tal como se ha mencionado con anterioridad, el Proyecto realizará de manera permanente
el mantenimiento y mejora de sus instalaciones y equipos, con el objetivo de permanecer
la mayor cantidad de tiempo funcionando en óptimas condiciones todas sus instalaciones,
por lo que no se puede predecir una fecha de inicio de esta fase. De ser necesario el cierre
o abandono se prevé como mínimo al año 30, con una duración de aproximadamente 10

meses.

# Mano de obra requerida

Se estima que, para desarrollar todas las actividades de la fase de cierre, se necesitará un
máximo de 25 personas.

# Provisión de suministros básicos

Tal como se ha mencionado con anterioridad, de ser necesario detener el funcionamiento

de la Planta de manera permanente, el desmantelamiento de las estructuras y oficinas será

realizado por empresas autorizadas del rubro que cuenten con experiencia en ello .
