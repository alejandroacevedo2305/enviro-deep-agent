El proyecto habitacional denominado “DIAGONAL LA ESTRELLA” corresponde a un Proyecto de
Edificación, acogido a la Ley de Copropiedad Inmobiliaria. Se configura a través de una agrupación
de edificios de departamentos, en altura de 7 pisos, en edificación continua y está emplazado en
un terreno con uso de carácter residencial, cuya superficie predial corresponde a 9.143,63 m2.

**térmico para viviendas nuevas del FSEV - RM”,** el cual evalúa complejos de techumbres, muros,
pisos ventilados, puertas opacas, sobrecimientos, elementos traslucidos, ventilación de recintos e
infiltración del aire.
