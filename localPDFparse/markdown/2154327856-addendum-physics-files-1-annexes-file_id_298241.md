# Diesel Ultra Copec

**Diesel Ultra** es un combustible formulado con aditivos

que aportan características para mejorar su

desempeño, generar ahorros de consumo, reducir

emisiones y evitar la formación de depósitos que

disminuyen la vida útil de los componentes del motor.
