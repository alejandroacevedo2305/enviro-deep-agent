**2.2.1** **Establecimiento del inicio de ejecución de proyecto**

Con la RCA aprobada se iniciará la confección de las obras civiles de
instauración de la planta de tratamiento, de modo de poder efectuar la

implementación de operación del sistema de tratamiento de RILes, tal como
muestra el cronograma presentado en la tabla anterior. Esta actividad es el inicio
de la ejecución de este proyecto.
