**1.2.4.- Modificación a un proyecto o actividad**

El Proyecto de “Bodegas Multiproposito de Sustancias Quimicas en instalaciones de inmobiliaria el Cardal
S.A.”, viene a modificar el proyecto “Bodegas de Almacenaje de Productos Químicos y Sustancias
Peligrosas”, que fue calificado ambientalmente favorable, mediante Resolución Exenta N°195, del 23 de
Octubre de 2012, de la Comisión de Evaluación de la región de Valparaíso.

La necesidad de este nuevo proyecto, obedece a que la actividad de servicios de bodegaje a terceros
requiere una mayor flexibilidad en el almacenaje de las cargas, pudiendo existir casos en que se requiera el
almacenaje a granel, que se requiera el almacenaje de una unica clase en las 7 bodegas o que se requiera
almacenar más de una clase en una misma bodega.

Acontinuación se detallan los cambios que modificarian los considerandos de las RCA No195/2012
