**Artículo 12.- Modificación del proyecto o actividad.**

El Proyecto consiste en la modificación de un Proyecto técnico aprobado por la Resolución de

Pesca N° 364/2006 y Resoluciones de Marina N° 647/2006 modificada por la N° 1515/2008. Las

modificaciones del proyecto consisten en el aumento de biomasa productiva desde 140 a 990

tons, y un total de 17 long-lines dobles de 200 mts.
