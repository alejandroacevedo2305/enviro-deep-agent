c. Para la disposición final se consideran estanques de polietileno LLDPE de acopio de aguas trata

d. Para la Fase de Construcción, el sistema de tratamiento de aguas servidas considerado son de

tipo modular prefabricado, las cuales consideran un sistema de tratamiento de lodos por aireación

extendida. El sistema de recolección contará con una red de tuberías sanitarias, las que conducirán

gravitacionalmente las aguas residuales desde los servicios higiénicos al Sistema de Tratamiento. El

afluente ingresará a una precámara de paso, donde las aguas serán filtradas de elementos sólidos

externos al proceso, tales como botellas plásticas, papel no biodegradable, etc. Posteriormente, el

afluente pasará a un tanque reactor donde se llevará a cabo el proceso de aireación, proceso en el

que las aguas permanecerán por un tiempo determinado, y con la ayuda de un soplador mecánico

permitirá la homogenización de las masas sólidas, evitando así la sedimentación de flóculos de ma
teria orgánica. Luego, pasarán a la fase de sedimentación, donde se separa el efluente depurado de

los lodos. Luego se dará lugar a la clarificación de las aguas servidas previamente tratadas, que
dando un sobrenadante que pasará a la fase de desinfección por medio de cloración y decloración.

Finalmente, el efluente tratado será dispuesto estanques para contener el agua tratada. Está pre
visto por diseño que la totalidad del agua sea utilizada en la humectación de caminos o frentes de

trabajo.

Para la Fase de Operación, se considera la implementación de un sistema de manejo de las aguas

servidas, compuesto por una fosa séptica que estará acoplada a baños. Su disposición final será

mediante drenes de infiltración al terreno. El sistema consistirá en 3 drenes de 4 metros de largo

cada uno.
