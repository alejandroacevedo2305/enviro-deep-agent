Declaración de Impacto Ambiental del proyecto "Proyecto Inmobiliario Loteo DFL N°2 con
construcción simultánea, Lote 4, Cerro Negro", contiene las observaciones generadas en virtud de la
revisión de la Adenda.

La respuesta a este Informe Consolidado deberá expresarse a través de un documento denominado
Adenda a la Declaración de Impacto Ambiental del proyecto "Proyecto Inmobiliario Loteo DFL
N°2 con construcción simultánea, Lote 4, Cerro Negro", la que deberá entregarse hasta el

16 de octubre de 2024.

comunicarlo por escrito a este Servicio, dentro del plazo que tiene para responder al Informe
Consolidado. Debe tenerse presente que posteriormente a esta fecha, se reanudará el proceso de
evaluación del proyecto.
