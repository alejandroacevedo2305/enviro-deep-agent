HABILITACIÓN DE TRANQUE Y RED DE CONDUCCIÓN INTRAPREDIAL

ROL PREDIO: 135-149

COMUNA: PAINE

1. DESCRIPCIÓN DEL PROYECTO

Proyecto considera la habilitación de tranque acumulador y la instalación de una red
de conducción, desde el tranque hacia distintos puntos del predio.

Con este proyecto se pretende dar seguridad de riego al predio, mediante la
habilitación del tranque, y aumentar las eficiencias de conducción y aplicación del
agua de riego. La primera en virtud de que se construirá una red de conducción,
eliminando la conducción por tierra desde el tranque y de un potrero a otro, y la
segunda dado que, dadas las diferencias topográficas, la red se habilitará desde el
tranque quedando presurizada para empalmar con algún sistema de riego que se
implemente en cada potrero.

El predio cuenta con aguas del canal Chada, que opera en régimen de turnos por lo
que es indispensable habilitar el tranque de manera de almacenar en él, las aguas
de cada turno. Luego, implementada la red de conducción, alimentar cada predio
de acuerdo, a las necesidades de riego.

2. DISEÑO DE LAS OBRAS

2.1.- DEMANDA DE AGUA

**a) Evapotranspiración de Referencia (ET0) (mm/dia)**

La ET0 se obtiene de la página web de la CNR (E-SIIR)

Para la zona del proyecto, corresponde a 5.43 mm/día o 0.63 l/s/hás

**b) Coeficiente de Cultivo**

Se escoge un valor KC = 1.15, de modo conservador, para no limitar el uso del sistema.

**c) Demanda neta y bruta (mm/dia)**

Demanda Neta = ET0 x Kc = 5.43 x 1.15

Demanda Neta = 6.25 mm/día

Considerando que la red quedará presurizada, se asume que se regará cada potrero
con un sistema de riego por goteo, luego la eficiencia de riego es = 0.9

Demanda Bruta = Demanda Neta / Eficiencia riego = 6.25 / 0.9

Demanda Bruta = 6.94 mm/día

2.1.- VOLUMEN DEL TRANQUE

Considerando 14.34 hás de riego, la demanda de agua será de:

Demanda Diaria Predio = 14.34 hás x 69.4 m3/día/hás = 995.20 m3/día

Considerando un turno de 7 días se requiere un volumen de almacenaje de:

Volumen Tranque = 7 x 995.20 = 6.966 m3

2.2.- CAUDAL DE DISEÑO

Considerando el riego de las 14.34 has en un tiempo de 12 horas, y dividiendo el
predio en 6 sectores dejando un margen de riego de 2 horas por cada sector, se
estima el caudal de diseño como sigue:

Q diseño = 69.400 l/hás x 2 hás / 2 hrs / 3.600 = 19.30 l/s

2.3.- DISEÑO DE LA RED DE CONDUCCIÓN

El diseño considera principalmente determinar el diámetro de tubería que cumpla
con las siguientes condiciones:

 La velocidad del agua en la tubería no debe superar el límite de 2.50 m/s

 La pérdida de carga debe ser mínima de manera de que se aseguré en la
entrada de cada sector la máxima presión posible dada la topografía del
terreno.

**1. DISEÑO DE SISTEMA DE RIEGO ROL 276-135**

Proyecto comprende un sistema de riego por goteo para 2.80 hás de frutales. Toda
esta superficie será plantada con naranjos los que requieren de sistema de riego
para poder ser regados apropiadamente.

1.1. DISEÑO AGRONOMICO

**a) Cultivo a regar:**

El cultivo a regar corresponde a naranjos.

**b) Evapotranspiración de Referencia (ET0) (mm/dia)**

La ET0 se obtiene ya obtenida que corresponde a 5.43 mm/día o 0.63
l/s/hás

**c) Coeficiente de Cultivo**

Se escoge un valor KC = 0.90

**d) Demanda neta y bruta (mm/dia)**

Demanda Neta = ET0 x Kc x Factor Cobertura = 5.43 x 0.90 x 0.90

Demanda Neta = 4.40 mm/dia

Demanda Bruta = Demanda Neta / Eficiencia riego = 4.40 / 0.90

Demanda Bruta = 4.90 mm/dia

**e) Tipo de riego y emisores**

Se implementará sistema de riego por goteo.

Para todos los sectores se considera línea de PE de 16 mm con gotero
integrado espaciados a 0.40 m.

**f) Marco de plantación**

Se considera el marco de plantación siguiente:

Entre hileras = 4.00 m

En la hilera = 3.50 m

**g) Marco de ubicación de emisores o su espaciamiento**

Los emisores se dispondrán en la hilera del cultivo cada 0.40 m. se
instalarán 2 líneas de goteo por cada hilera de cultivo.

Corresponde un marco de emisor (2) de 0.4 x 4.00

**h) Caudal del emisor**

Se utilizará una línea de goteo Hidrogol 3.9/0.40 m. El caudal del emisor es
de 3.90 l/hr.

**i) Precipitación del sistema:**

PP sistema = N° emisores x Caudal Emisor / Marco Emisores

PP sistema = ( 4 x 3.90 ) / ( 0.4 x 4.0 ) = 9.75 l/m2/hr = 9.75 mm/hr

**j) N° de laterales por hilera de cultivo**

Se consideran 2 laterales por hilera de cultivo.

**k) Tiempo de riego Sector:**

Todos los sectores tendrán la misma disposición y tipo de emisor, por tanto
todos los sectores tienen el mismo tiempo de riego:

Tiempo riego= Lámina Bruta x Marco Plantación / (Caudal Emisor x N° Em.

por planta)

Tiempo Riego = 4.90 x 3.50 x 4.00 / (3.90 x 18) = 0.97 horas= 1 hora.

**l) N° de Sectores o bloques de riego**

Tiempo Total Disponible para riego con bomba= 18 horas

Nro máximo de Sectores = Tiempo Total Disponible / Tiempo de Riego
Sector

Nro máximo de Sectores = 18 /1.00 = 18 sectores

Se considera dividir los potreros existentes en sectores de máximo 0.50
hás, Finalmente se divide el riego en 5 sectores.
