# CALICATAS

Código Proyecto : GTS-PY14013-0002

Nombre Proyecto : Obras Civiles Fase 7, Los Bronces

Código documento : GTS-P14013-0002-20IT-002

**1** **Prospecciones mediante calicatas**

El día miércoles 1 de Abril se realizaron las excavaciones de las calicatas GTS-001 y GTS-002,
con fin de determinar características geotécnicas del terreno mediante la inspección visual de
éste (Descripción Estratigráfica) y posteriormente tomar muestras representativas del material
para ser analizadas de acuerdo al programa de laboratorio.

A continuación se presenta la ubicación de las calicatas ejecutadas.

Figura 1: Ubicación Calicatas

**2** **Estratigrafía Visual**

A partir de las dos calicatas realizadas se elaboró una estratigrafía general del sector, en la
cual se pueden apreciar dos horizontes bien definidos con las siguientes características:

**Horizonte 1:** Corresponden a gravas sub redondeadas y sub angulares en una matriz
arenosa con tamaño máximo de 4”, no plásticas. Humedad media y compacidad moderada.
Espesor de estrato 30 cm.

**Horizonte 2** : Corresponden a gravas sub redondeadas y sub angulares con bloques
mayores a 6” y arenas, de plasticidad media. Humedad media y compacidad débil a moderada.
Espesor del estrato 1,9 m.
