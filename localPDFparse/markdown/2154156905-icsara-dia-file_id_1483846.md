mínimas que aseguren el cumplimiento de su propósito.

descripción de suelo del área comprometida, lo que necesariamente requiere descripción a
través de calicatas y el uso histórico del entorno, elementos necesarios para demostrar que
existen limitaciones productivas reales.

proyecto fotovoltaico, así como el incremental que se produciría por la ejecución del CAV
