# CAPÍTULO 2 “Modificación de un proyecto o actividad” Ampliación del volumen de extracción y procesamiento de áridos Planta El Carmen

Áridos Misael Elcías Sáez Rivas EIRL

DECLARACIÓN DE IMPACTO AMBIENTAL - CAPÍTULO 2

**CAPÍTULO 2. MODIFICACIÓN DE UN PROYECTO O ACTIVIDAD**

Según el Artículo 12° del Reglamento el Sistema de Evaluación de Impacto Ambiental,
D.S.40/2013 modificado mediante D.S. N°30/2024, se señala que:

“El Titular deberá indicar si el proyecto o actividad sometido a evaluación modifica un proyecto
o actividad. Además, en caso de ser aplicable, deberá indicar las Resoluciones de Calificación
Ambiental del proyecto o actividad que se verán modificadas, indicando de qué forma.

En caso de modificarse un proyecto o actividad, la calificación ambiental deberá recaer sobre
dicha modificación y no sobre el proyecto o actividad existente, aunque la evaluación de
impacto ambiental considerará la suma de los impactos provocados por la modificación y el
proyecto o actividad existente para todos los fines legales pertinentes”.

Según lo anterior, el Titular señala que el Proyecto “Ampliación del volumen de extracción y
procesamiento de áridos Planta El Carmen” no modifica algún proyecto ni alguna actividad con
Resolución de Calificación Ambiental.
