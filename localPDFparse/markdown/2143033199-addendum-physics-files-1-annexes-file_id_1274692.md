|**Partes, Obras y Acciones**|La construcción del parque fotovoltaico considera las siguientes instalaciones:<br>**- Instalación de Faenas**; Instalaciones que permiten ejecutar los trabajos<br>constructivos. Se contempla una instalación de faenas de superficie<br>aproximada de 2.900 m2. Se utilizará esta superficie para instalar oficinas,<br>bodegas, estacionamientos, patios de acopios, patios de residuos, etc. Estas<br>instalaciones serán del tipo modulares móviles, tipo contenedor 20/40 pies o<br>similar.
