**1.3.11** **Descripción Cronológica de sus Fases**

A continuación, se presenta una breve descripción de las actividades asociadas a cada fase del

Proyecto.

- **FASE 1: CONSTRUCCIÓN**

Esta etapa contempla las siguientes actividades: Habilitación de instalaciones de apoyo,

construcción de cerco perimetral, montaje mecánico y eléctrico, instalación de redes de baja

tensión (BT) y media tensión (MT), conexión a red MT, y posteriormente las pruebas y puesta en

marcha del parque solar.

- **FASE 2: OPERACIÓN Y MANTENCIÓN**

Esta etapa contempla el funcionamiento de todas las componentes del parque, incluyendo la

operación de éste, como también sus mantenciones programadas y no programadas.

- **FASE 3: CIERRE**

Consiste en desmantelar las obras físicas y restituir las condiciones del emplazamiento, llegando a

lograr el mismo aspecto que la zona tenía con anterioridad al desarrollo y operación del Proyecto.
