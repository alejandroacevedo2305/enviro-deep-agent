**2.5** **APLICABILIDAD ARTICULO 16, ESTABLECIMIENTO DEL INICIO DE EJECUCIÓN DE**

**PROYECTO**

El proyecto **“Regularización y continuidad planta de asfalto e implementación de**
**extracción y procesamiento de áridos”,** dará inicio a sus actividades con la preparación del
terreno, que abarca las labores de demarcación, implementación de señalética y despeje de las
cuñas a extraer; mientras la planta de asfalto continua operando de manera simultánea, dando
cumplimiento a la RCA N° 212 del proyecto original.
