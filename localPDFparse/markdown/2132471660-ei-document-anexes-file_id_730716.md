## Laboratorio Viña del Mar

**3.0** **EXPLORACION DEL TERRENO**

Para poder desarrollar los trabajos encomendados se procedió con el siguiente programa de
exploraciones:

**Calicatas:** se confeccionaran 10 calicatas de 2.5 m de profundidad.

**Descripción visual:** en el proceso de ejecución de calicatas se realizará una descripción

visual de los estratos que se detecten.

**Densidad in situ:** se controlaran densidades in situ de cada calicata. Se consulta el

control de 8 densidades in situ por cada calicata.

**Cono dinámico:** dado que se prevé, que existan rellenos de mayores profundidades que

2.5 m, se incluye la ejecución de 3 conos dinámicos, hasta una
profundidad máxima de 6 m o antes si se presenta el rechazo.

**Ensayos de laboratorio:** de las muestras representativas de cada calicata, se realizarán

los siguientes ensayos: análisis granulométrico, límites de Atterberg,
clasificación de suelos por el método USCS y AASHTO, análisis
Proctor y capacidad de soporte California. Para ello se considera un

total de 20 muestras a analizar.
