# Camarico Solar Establecimiento del inicio de la ejecución del Proyecto

La gestión, acto o faena mínima que inicia la ejecución del Proyecto, de modo

sistemático y permanente, de acuerdo con su tipología, es el montaje de las

instalaciones temporales dando cuenta de manera adecuada el inicio del

Proyecto para comenzar con la construcción del proyecto.

Se proyecta su inicio en febrero del 2022.
