El Caudal máximo se determina con la siguiente expresión:

Q máx. = C x I x A

**Determinación de la Precipitación para P.R.=100 años**

I = 1,1 x CF x CD x P 24 [10 ]

2

CF : Coeficiente de frecuencia (1,35)
CD : Coeficiente para lluvias de una hora de duración (0,17)
P 24 [10] : Precipitación diaria para período de retorno de 10 años (107,3)

Los valores de CF, CD y P 24 [10] se obtuvieron de la Guía de Diseño “Técnicas Alternativas para Soluciones
de Aguas Lluvias en Sectores Urbanos”

I = 1,1 x 1,35 x 0,17 x 107,3 = 27,1 mm/hr

Con estos valores el Q máx. es:

Q máx. = 0,70 x 27,1 x 0,011647 = 0,0614 m [3] /seg = 61,4 lt/seg
