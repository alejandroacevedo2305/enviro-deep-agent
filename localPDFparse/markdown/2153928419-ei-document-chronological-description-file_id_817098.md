**1.8.3 Mano de obra de la fase de construcción**

Para la construcción del Proyecto, se prevé una dotación máxima de 150 trabajadores, sin embargo, se

considera una dotación promedio de 75 personas.

Para la determinación de residuos (domiciliarios y líquidos), asociados al personal de obra, se considera la

dotación máxima (150 personas).

Se contempla un total de 20 unidades de estacionamientos de vehículos para los trabajadores y, además, se

instalarán bicicleteros para fomentar el uso de bicicletas a los trabajadores.
