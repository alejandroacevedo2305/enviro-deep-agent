DIA - “CENTRO DE ACONDICIONAMIENTO Y EXPORTACION DE SEMILLAS PLANTA BULNES”

3.4. DESCRIPCIÓN DE SUS PARTES, ACCIONES Y OBRAS QUE LO COMPONEN.

El proyecto contempla las siguientes etapas:

Fase Construcción: Instalación de faenas, edificación, infraestructura eléctrica,
sanitaria y obras generales.

Fase Operación: Habilitación del proyecto denominado “Centro de
Acondicionamiento y Exportación de Semillas planta Bulnes”.

Fase de Abandono: Este proyecto no contempla etapa de cierre o abandono,
ya que se trata de un proyecto que por sus características tiene una vida útil
indefinida. Sin embargo, se establecerá un límite de 50 años para efecto del
procedimiento de Evaluación de Impacto Ambiental.

El programa de actividades general se realizará de acuerdo a lo establecido en la
siguiente tabla:
