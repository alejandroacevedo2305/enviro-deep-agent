**Descripción General**

Dimensiones (m2) Largo Ancho

Descripción del Sitio

Conjunto disperso de materiales históricos entre los que se identifican el fragmento de una botella de vidrio de manufactura
industrial y procedencia escocesa. Se registran dos fragmentos de loza importada del siglo XIX, uno de los cuales
corresponde al borde de un plato con decoración estilo Willow, mientras que el otro corresponde al fragmento de una taza
con decoración anular. Se registran dos fragmentos cerámicos monócromos de probable cronología histórica. Se registra
también el fragmento de un aislante eléctrico de gres.
