# DESCRIPCIÓN CRONOLÓGICA DE SUS FASES

Respecto de los contenidos mínimos de las Declaraciones de Impacto Ambiental establecidos
en el DS 40/12 RSEIA, en este apartado se presenta el cronograma del Proyecto.

Una vez que la Resolución de Calificación Ambiental haya sido emitida (mes 0), se procederá
a la Etapa de Construcción, para la cual se estima una duración de 2 meses, mientras que
para la Etapa de Operación, se estima una duración de 20 años. El programa de actividades
se detalla en el siguiente cronograma:
