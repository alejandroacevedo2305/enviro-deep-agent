#### 1.- Definición del Sistema:

El sistema desarrollado para la Industria Salmonera tiene como
objetivo realizar el ensilaje de la mortalidad de los centros de
cultivo.
El sistema de ensilaje corresponde a un procedimiento de
transformación de la mortalidad, mediante una molienda y
adición de acido fórmico hasta alcanzar y mantener un pH entre
3,5 y 4, en una mezcla homogénea y estable.

#### 2.-Objetivo:

Transformar un producto perecible y contaminante, en un
material inerte que no contamina, que no emite olores y que
controla tempranamente mediante molienda y acidificación la
actividad biológica que pudiera generar contaminación.

#### 3.- Alcance:

El contenido de este instructivo comprende el proceso que se
aplica para todas las mortalidades frescas como también la
operación de los Equipos de Ensilaje, marca Global Fluid &
Landia, diseñados para este fin.

#### 8.- Tablas para el uso de Acido Fórmico:

**b) Dosificación y Aplicación del** **Acido Fórmico** (Acido al 60% de concentración)

Se requiere un proceso continuo para medir el PH y dosificar el acido. El
estándar internacional de PH para ensilaje es entre 3,5 y 4.

Cálculo de la Dosis de Acido Fórmico.

Acido Fórmico en kg. = Mortalidad en kg. x dosis de acido (típico)

(Dosis de acido comúnmente usado para ensilaje: 3,5%)

Densidad de acido fórmico: 1,22 gr./ml

Acido (lts.) = Mortalidad (kgs) x Densidad x Dosis

Concentración del acido
Por ejemplo:
200 kg x 1,22 kg/lt x 0,035 : 0,60 = 14,23 litros (de acido al 60%)
