**4.2** **Restitución Aero fotogramétrica digital**

suministradas. La cantidad de puntos de unión determinados por el software de restitución digital dependió directamente de la

calidad de las imágenes. La malla de Aero-triangulación demuestra la alta calidad de las imágenes y la robusta correlación de

elementos entre fotogramas obtenidos con el sensor incorporado en el equipo INSPIRE 1.
