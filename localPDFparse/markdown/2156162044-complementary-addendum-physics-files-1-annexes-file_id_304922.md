# 1 INTRODUCCIÓN

La presente Memoria de Cálculo se refiere a las Obras de Mitigación de Aguas lluvias por los trabajos a

realizar para el proyecto de “Condominio Parque Constanera”, a emplazarse en la comuna de Temuco,

Región de La Araucanía.

**FIGURA 1: UBICACIÓN PROYECTO CONDOMINIO PARQUE COSTANERA.**

Esta Memoria se complementa con los planos del proyecto.

El proyecto contempla la captación del agua precipitada sobre un área que incluye todas las cubiertas de
jardines y pavimentos del condominio ya mencionado. Las aguas lluvia captadas serán conducidas por
colectores que permitirán evacuarlas hacia drenes dentro del proyecto.

# 2 DISEÑO

Considerando el área aportante y el coeficiente de escorrentía correspondiente, con la Fórmula del
Método Racional, se obtiene el caudal de diseño.

= _C_ ⋅ _i_ ⋅ _A_  _m_ 3 
_Q_ 3, 6 ⋅10 6  _s_ 

Donde:

C: Coeficiente de Escorrentía (0,95 para techumbre, 0,88 para pavimento y 0,25 para tierra o área
verde)
i: Intensidad de la lluvia de diseño, (mm/h).
A: Área Aportante.
