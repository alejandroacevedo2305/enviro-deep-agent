Fuente: Elaboración propia.

Se dará inicio a la implementación del Proyecto con la instalación de faenas, en
septiembre de 2015 o una vez obtenida la Resolución de Calificación Ambiental (RCA)
favorable.

Se estima que se dará inicio a la Etapa de Operación en octubre del 2016; una vez
concluida las pruebas de pre-entrega de las obras. Respecto a las medidas de atenuación
acústicas tendrán una vida útil indefinida con la adecuada operación y mantención de sus

instalaciones.

Dada las características del Proyecto, no se considera una etapa de cierre y abandono ya
que las obras de atenuación acústicas formarán parte de las estructuras del Viaducto; las
cuales tendrán una vida útil indefinida.
