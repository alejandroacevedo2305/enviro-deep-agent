**1. Introducción**

A continuación, se presenta la descripción de las rutas de transporte terrestre que serán

utilizadas por vehículos pesados y livianos para el transporte de insumos, equipos, residuos,

mantención y mano de obra requeridos para la materialización del Proyecto “Energía

Esmeralda” ubicado en la comuna de Quillota, Región de Valparaíso.
