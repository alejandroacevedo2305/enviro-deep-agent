## 1 DESCRIPCIÓN DEL PROYECTO

**Instalaciones**

**1.** **El Proponente indica en Adenda N°1 que:** _**“La compactación se efectúa a humedad**_

_**óptima, precediéndose a regar las veces que sea necesario, hasta alcanzar un**_
_**Proctor Modificado de como mínimo del 95%”**_ **, al respecto, el Proponente deberá**
**establecer y justificar cual es el parámetro de “humedad óptima” y el medio de**
**verificación a utilizar para asegurar el cumplimiento de dicha humedad.**

**RESPUESTA:**

La humedad óptima es aquella humedad con la cual el suelo alcanza su máxima densidad. Al
realizar el ensayo Proctor Modificado se van compactando diferentes muestras del mismo terreno
a las que se le agrega una cantidad de agua distinta, evaluando que porcentaje de humedad es el
que permite la máxima compactación. Aquella que queda con mayor densidad se le denomina
Proctor 100%, y la humedad de esta muestra es la humedad óptima.

Se asegura por tanto, que la humectación del terreno se realizará para que el terreno tenga
humedades que permiten un mínimo del 95% de su compactación máxima. Para verificar que el
terreno contiene la humedad necesaria se realizaran ensayos in situ como el método del Cono de

Arena.

La información técnica de los cálculos se presenta a continuación:

Para el coeficiente de infiltración (K) primero se realizó la prueba de absorción del terreno,
utilizando las instrucciones del Reglamento General de Alcantarillados particulares. Para ello se
realizó una calicata cúbica de 30x30 cm, en la zona destinada para la infiltración, a una
profundidad mínima de 1 metro, se llenó con agua hasta una altura de 15 centímetros, y se dejó
que ésta filtrará totalmente. Posteriormente, se colocó agua nuevamente y se tomó el tiempo
medio que tarda en bajar 2,5 centímetros; con este tiempo se observó la tabla en el cual se
determina el Índice de Absorción. Posteriormente se verificó por el método para la comprobación
de la prueba anterior. El tiempo medio que tardó en bajar 2,5 centímetros fue de 4 minutos, lo
que implica que el índice de absorción es de 135 lt/m [2] /día.

Para obtener la superficie necesaria para infiltrar el efluente de la fosa séptica, se realiza el
siguiente cálculo:

**S = Q/K**

**Donde:**
**S: Área de infiltración, en m** **[2]** **.**
**Q: Caudal total de aguas a infiltrar (30.000 l/d construcción; 400 l/d operación)**
**K: Coeficiente absorción para drenes en l/m** **[2]** **·día (135/m** **[2]** **/día).**

Utilizando la ecuación para área de infiltración y para las características del suelo (K= 135 l/m [2] /día)
del área de emplazamiento del sistema de absorción, se obtiene un área de infiltración igual 222,2
m [2] para la fase de construcción y de 2,96 m [2] para la fase de operación.

Utilizando drenes de infiltración, el largo de éstos se puede calcular de la siguiente forma:

L = Q/ A*K

Donde:

L: Largo de la zanja en metros.
A: Ancho de la zanja en metros. Se emplea A: 1,2 m

Utilizando la ecuación para calcular el largo de la zanja de infiltración y para las características del
suelo (K= 135 lt/m [2] /día) del área de emplazamiento del sistema de absorción, se obtiene que el
largo del sistema de drenaje es de 185,2 m para la fase de operación y de 2,5 m para la fase de
operación.
