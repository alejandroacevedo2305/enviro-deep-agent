# 6 COMPROMISOS AMBIENTALES VOLUNTARIOS

Para efectos del presente Proyecto, ENAP - Magallanes asume los siguientes compromisos
(Tabla 6-1).

<!-- # 6 COMPROMISOS AMBIENTALES VOLUNTARIOS Para efectos del presente Proyecto, ENAP - Magallanes asume los siguientes compromisos (Tabla 6-1). -->

**Tabla 6-1: ****

| COMPROMISO AMBIENTAL<br>VOLUNTARIO | INFORMACIÓN ESTRATIGRÁFICA E IDENTIFICACIÓN Y<br>MEDIDAS DE PROTECCIÓN DE ACUÍFEROS |
| --- | --- |
| **IMPACTO ASOCIADO** | Afectación de Acuíferos. |
| **FASE DEL PROYECTO** | Operación - Fracturación. |
| **DESCRIPCIÓN DEL COMPROMISO** | Una vez perforado cada pozo, y previo a cada fracturación, se<br>entregará información estratigráfica que permita reconocer los<br>acuíferos intervenidos con su ejecución, sus niveles y las<br>características, clasificación y estratigrafía de los materiales que<br>componen su matriz y los mantos o estratos como también la<br>identificación y medidas de protección de acuíferos para cada<br>pozo. |
| **JUSTIFICACIÓN DEL**<br>**COMPROMISO** | Protección de acuíferos. |
| **LUGAR DE IMPLEMENTACIÓN DEL**<br>**COMPROMISO** | Área de fracturación de cada pozo. |
| **FORMA DE IMPLEMENTACIÓN**<br>**DEL COMPROMISO** | Durante la fracturación de cada pozo. |
| **OPORTUNIDAD DE**<br>**IMPLEMENTACIÓN DEL**<br>**COMPROMISO** | Una vez perforado cada pozo y previo a la fracturación de cada<br>uno de ellos. |
| **INDICADOR QUE ACREDITE SU**<br>**CUMPLIMIENTO** | No alternación de acuífero. |
| **FORMA DE CONTROL Y**<br>**SEGUIMIENTO**<br> | Registro en la Superintendencia del Medio Ambiente (SMA). |
