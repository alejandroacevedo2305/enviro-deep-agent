**DESCRIPCIÓN DEL PRODUCTO**

NALCO 9986 es un polvo de polímero catiónico
de alto peso molecular.
NALCO 9986 está diseñado para la
deshidratación y espesamiento de lodos,
clarificación de agua cruda y efluentes. Este
producto mejora la eliminación de sólidos en
suspensión en un número diverso de
aplicaciones industriales y municipales.
Este producto funciona bien con todo tipo de
equipo que incluye filtros de prensa de banda
doble, prensas de tornillo, flotación por aire
flotación por aire disuelto / inducido, filtro
prensa de placas y marcos, filtros de prensa de
vacío también como aplicaciones de alto

cizallamiento.

**SEGURIDAD Y MANEJO**

Como todos los productos químicos, NALCO
9986 debe manejarse con cuidado.
Antes de usar el producto, consulte la hoja de
datos de Seguridad (FISPQ), sección Control de
exposición y Protección Personal, para indicación
de EPP e identificación de peligros, para
indicaciones sobre exposición y contacto.
Evite el contacto con la piel, ojos y ropa. No
ingerir. Utilizar con ventilación adecuada.
