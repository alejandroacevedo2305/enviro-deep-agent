# DESCRIPCIÓN CRONOLÓGICA DE LAS FASES DEL PROYECTO

de 2021. La fase de operación tendrá una duración aproximada de 11 meses, dentro de un período de 12 meses desde la aprobación, estimándose

su inicio también marzo de 2021. Finalmente, la fase de cierre tendrá una duración aproximada de 15 días, estimándose para esto el mes de febrero
