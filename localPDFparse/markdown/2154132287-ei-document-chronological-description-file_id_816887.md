**A.6.2.** **Cronología**

Se ha estimado que el inicio de la Fase de construcción del Proyecto se realice en **Marzo 2022**, considerando
una duración aproximada de 32 meses para Vicente Valdés I y 31 meses para Vicente Valdés II, en su totalidad.

la Fase de construcción corresponde a la Recepción Municipal.
