# SET FOTOGRÁFICO

## EDIFICIO CONDE DEL MAULE 4470

### MHO CONSULTORES ASOCIADOS LTDA .

Santiago, Septiembre de 2015

**Set Fotográfico**

El presente anexo contiene fotografías capturadas en visitas a terreno efectuadas durante los días 07 de
julio y 09 de septiembre del 2015. En él se da a conocer el estado actual de los terrenos dónde se
emplazará el proyecto “Edificio Conde del Maule 4470” (en adelante “Proyecto”), se describe además el
estado de las calles y la existencia de proyecto similares (en evaluación y ejecutados) en el área de
influencia.

La figura 1 muestra la ubicación geográfica del proyecto, indicando los puntos donde fueron realizadas las
fotografías.
