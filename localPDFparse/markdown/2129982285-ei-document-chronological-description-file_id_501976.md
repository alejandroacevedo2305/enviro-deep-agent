**3.6.3. Cronograma de las Principales Obras, Partes y Acciones de la Fase de**
**Operación (Art. 19o a.6.3)**

La vida útil del proyecto se estima en 50 años. No obstante, debido a la tipología del
Proyecto y realizando un Plan de Mantención adecuado, en donde se considere el
reemplazo de las unidades de generación asociado a los avances tecnológicos, es posible
que esta vida útil se prolongue, situación que, de ocurrir, será debidamente informada a
los organismos sectoriales atingentes.
