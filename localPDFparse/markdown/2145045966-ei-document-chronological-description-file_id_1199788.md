Declaración de Impacto Ambiental
Proyecto Inmobiliario Los Conquistadores

Constructora Malpo SpA.

**Cronograma de las fases del proyecto**

**Cronograma de la fase de construcción**

El proyecto se desarrollará en 3 fases constructivas parcialmente paralelas. Las primeras

dos fases constructivas, tendrán una duración estimada de 15 meses, mientras que la

tercera (última) fase constructiva, tendrá una duración estimada de 11 meses, estimándose

un periodo total de 3 años para el desarrollo total del proyecto.

**Tabla 1. Fechas estimadas de inicio y término para cada fase constructiva.**

|Fase constructiva|N° Viviendas|Fecha de inicio (*)|Fecha de término|Duración estimada|
|---|---|---|---|---|
|1|152|Octubre 2020|Diciembre 2021|15 meses|
|2|129|Octubre 2021|Diciembre 2022|15 meses|
|3|93|Noviembre 2022|Septiembre 2023|11 meses|

_(*) El inicio de las fases está sujeto a la obtención de la Resolución de Calificación Ambiental (RCA) favorable._

**Cronograma de la fase de operación**

Tal como se ha expuesto en los párrafos anteriores, esta fase se considera como indefinida

y, debido a que se contemplan 3 fases constructivas para el proyecto, la fase de operación

se fragmenta de la misma forma, por lo que el inicio de cada fase de operación queda sujeto

al término de la respectiva fase constructiva.

A continuación, se presentan las fechas estimadas del inicio de las fases de operación:

**Cronograma de la fase de cierre o abandono**

El proyecto no contempla fase de cierre o abandono, ya que se trata de un proyecto
