# DESCRIPCIÓN CRONOLÓGICA DE SUS FASES

Una vez que la RCA haya sido emitida (día 0), se procederá a ejecutar, para cada pozo a fracturar, la Etapa de Montaje, para la
cual se estima una duración máxima de 14 días, para la Etapa de Operación se estima una duración de 20 días y para la Etapa
de Desmontaje, una duración de 1 día.

A continuación, se presenta una tabla que resume lo señalado para cada pozo a fracturar, contemplando una duración total de

35 días.
