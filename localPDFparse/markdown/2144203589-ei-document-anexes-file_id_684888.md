# 1. INTRODUCCIÓN

De acuerdo a lo que establece el Título III, Párrafo 1, artículo 12 del D.S. No40 del Ministerio del
Medio Ambiente, el cual aprueba el Reglamento del Sistema de Evaluación del Impacto Ambiental,
indica dentro de los contenidos mínimos que deben contar las Declaraciones de Impacto Ambiental,
el Titular debe indicar si el proyecto o actividad sometido a evaluación modifica un proyecto o
actividad.

Además, en caso de ser aplicable, deberá indicar las Resoluciones de Calificación Ambiental del
proyecto o actividad que se verán modificadas, indicando de qué forma.

En caso de modificarse un proyecto o actividad, la calificación ambiental deberá recaer sobre dicha
modificación y no sobre el proyecto o actividad existente, aunque la evaluación de impacto
ambiental considerará la suma de los impactos provocados por la modificación y el proyecto o
actividad existente para todos los fines legales pertinentes.

# 2. MODIFICACIÓN DE UN PROYECTO O ACTIVIDAD

El Titular indica que el presente Proyecto corresponde a un proyecto nuevo, por lo tanto, no se
trata de una modificación de algún proyecto con RCA ya existente.
