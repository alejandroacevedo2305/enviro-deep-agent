**- Limpieza de redes para arrastre de elementos sólidos**

Este tipo de limpieza se aplica cuando existen problemas de turbiedad en
redes de baja o alta circulación, con presencia de hierro y/o manganeso,
sólidos de cualquier índole, o en instalaciones nuevas.

exista arrastre, la velocidad del agua en las tuberías debe superar los 0,5
m/s, dado que las velocidades menores pueden provocar sedimentación en
la tubería.

indeseables, lo que puede ocurrir en tuberías nuevas durante su
instalación.

 - Se debe realizar un lavado, como se especifica en el punto anterior,

haciendo un arrastre de sólidos.

 - Si la red es de una urbanización nueva, este trabajo es del urbanizador

y Nuevosur solo verificará que tanto las desinfecciones como los
lavados sean realizados a cabalidad.

 - Se sectoriza el tramo a desinfectar.

 - Se aplica 250 ml de hipoclorito de sodio al 10% por cada metro cúbico

de agua contenida en las redes (25 mg/l). La aplicación se realiza por
tramos, cada 50 metros o bien durante la incorporación de las aguas a
las tuberías de la red. Esta aplicación deberá hacerse con camión
aljibe o inyecciones en la tubería.

 - La tubería debe quedar completamente llena para que la desinfección

sea eficiente.

 - Se deja actuar por 24 horas.

 - Se descartan todas las aguas, de ser posible usando mangueras, en

grifos y desagües.

 - Se procede a realizar un segundo lavado tipo renovación de agua,

siguiendo todo el proceso descrito en este documento, hasta
estabilizar el cloro residual, que puede alcanzar valores altos, y la
turbiedad. Se procederá a tomar muestra bacteriológica, las que serán
analizadas por el laboratorio correspondiente.
