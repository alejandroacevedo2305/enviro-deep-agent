# DECLARACIÓN DE IMPACTO AMBIENTAL Extracción Procesamiento de Áridos Táfra y CAPÍTULO VIII RELACIÓN CON LAS POLÍTICAS Y PLANES EVALUADOS ESTRATÉGICAMENTE

CAPÍTULO VIII: RELACIÓN CON LAS POLITICAS Y PLANES EVALUADOS ESTRATEGICAMENTE

JULIO, 2014

**RELACIÓN CON LAS POLÍTICAS Y PLANES EVALUADOS ESTRATÉGICAMENTE**

En cumplimiento al Artículo 15 del DS 40/2012 Reglamento del Sistema de Evaluación
de Impacto Ambiental del MMA, a continuación se identifican las políticas y planes
evaluados estratégicamente atingentes al Proyecto, así como la compatibilidad de éste
con el uso del territorio y los objetivos ambientales de tales planes y políticas.

**IDENTIFICACIÓN Y RELACIÓN CON LAS POLÍTICAS Y PLANES EVALUADOS**
**ESTRATÉGICAMENTE, A NIVEL REGIONAL**

Con respecto a la Región de Magallanes y Antártica Chilena, esta no cuenta con
políticas y planes evaluados ambientalmente a la fecha.

**IDENTIFICACIÓN Y RELACIÓN CON LAS POLÍTICAS Y PLANES EVALUADOS**
**ESTRATÉGICAMENTE, A NIVEL COMUNAL**

Con respecto a la comuna de Primavera, esta no cuenta con políticas y planes
evaluados ambientalmente a la fecha.
