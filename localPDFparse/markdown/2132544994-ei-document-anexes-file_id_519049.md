# Cotización Arriendo

## 1 Alisadora de pavimento rotatoria marca Stow 36” accionada con motor Honda 5,5 HP.

Características:

 Equipada con 4 aspas de 6" x 14" con un diámetro de trabajo de 36" (90 cm).

 Accionado con un motor a gasolina, marca Honda de 5,5 HP, de 4 tiempos.

 Peso de operación: 75 kg.

 La inclinación de las paletas es regulable, por medio de un sistema de ajuste
rápido.
