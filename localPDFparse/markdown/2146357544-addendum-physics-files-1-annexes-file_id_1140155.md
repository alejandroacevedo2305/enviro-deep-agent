# ESTIMACIÓN DE EMISIONES

**Estimación de emisiones**

En el presente informe se presentan las estimaciones de emisiones atmosféricas generadas durante
las etapas de construcción y operación del proyecto en evaluación **“REGULARIZACIÓN BODEGA DE**
**VINOS Y SISTEMA DE TRATAMIENTO DE RILES, VITIVINÍCOLA INVINA LIMITADA”**, en adelante

“Proyecto”.

Para estimar las emisiones atmosféricas generadas por el proyecto, se realizó aplicando los factores

de emisión y formulas propuestas por la Guía para la Estimación Atmosférica de Proyectos

Inmobiliarios, de la SEREMI Región Metropolitana, del Ministerio del Medio Ambiente del año 2012

**1.Estimación de emisiones Etapa de construcción Obras existentes**

**1.1. Escarpe**

Esta actividad corresponde a la preparación del terreno utilizado para el emplazamiento de las

obras existentes. Para esto, se realizó una nivelación de 3 hectáreas.

Para poder calcular las emisiones respectivas para esta actividad, es necesario calcular cual es la

distancia que recorre el cargador, considerando el área mencionada anteriormente. La guía

presenta un rendimiento general de 3,57 km/ha. Considerando esto, se tiene que:

Distancia Recorrida= 3 (há)*3,57(km/há) = 2.5 Km

De esta forma y considerando el factor de emisión mencionado en la tabla 1, se obtiene lo

siguiente:
