Declaración de Impacto Ambiental del proyecto **"Parque Fotovoltaico Zaldivar"**, contiene las

Adenda a la Declaración de Impacto Ambiental del proyecto **"Parque Fotovoltaico Zaldívar"**, la
que deberá entregarse hasta el 22 de julio de 2024.

comunicarlo por escrito a este Servicio, dentro del plazo que tiene para responder al Informe
Consolidado. Debe tenerse presente que posteriormente a esta fecha, se reanudará el proceso de
evaluación del proyecto.
