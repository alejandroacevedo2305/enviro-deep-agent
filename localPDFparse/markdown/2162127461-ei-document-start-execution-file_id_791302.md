Declaración de Impacto Ambiental

Proyecto “Regularización y Ampliación Parque Industrial- Maule”

Inmobiliaria y Comercial SOTEXI Ltda.

**1.2.10** **Establecimiento del inicio de ejecución del Proyecto**

El Proyecto sometido a evaluación ambiental está compuesto por una fase existente la cual

se encuentra operativa por cuanto no se considera una acción que marque el inicio de

ejecución de esta. Por otro lado, en cuanto a la fase constructiva correspondiente a la

ampliación del Proyecto, la actividad que marcará el inicio de esta fase será la instalación de

faenas.
