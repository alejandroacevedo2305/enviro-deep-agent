# 1 INTRODUCCIÓN

A continuación, se presenta la descripción pormenorizada de las acciones de transporte de carga

terrestre mediante la utilización de vehículos pesados y livianos para el transporte de equipos,

residuos, suministros e insumos básicos requeridos para la materialización del Proyecto **“Parque**

**Fotovoltaico El Loro Choroy” durante su fase de construcción** (Tabla 1-1) **,** ubicado en la comuna de

Curicó, Provincia de Curicó, Región del Maule (Figura 1-1). El proyecto consiste en la instalación,

operación y eventual cierre de una central solar fotovoltaica de 12 MWp de potencia instalada.
