DIA “ _Regularización_ _Planta de Tratamiento de Residuos Industriales Líquidos, Delisur”_

**6.** **COMPROMISOS AMBIENTALES VOLUNTARIOS**

Según el artículo 19-e) del Reglamento del Sistema de Evaluación de Impacto Ambiental,

el titular del proyecto deberá describir el contenido de aquellos compromisos ambientales

voluntarios, no exigidos por la legislación vigente, pero que se deseen llevar a cabo en

alguna de las etapas del proyecto.

Al respecto el titular del proyecto “Regularización Planta de Tratamiento de Residuos

Industriales Líquidos, Delisur” no tiene considerado adoptar compromisos ambientales

voluntarios para este proyecto.
