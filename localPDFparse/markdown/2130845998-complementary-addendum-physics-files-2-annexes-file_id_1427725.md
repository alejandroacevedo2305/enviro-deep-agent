**MEMORIA DESCRIPTIVA SISTEMAS DE DETECCIÓN Y COMBATE DE INCENDIOS**

**PROYECTO PLANTA RILES**

**I.** **OBJETIVO**

Describir componentes y funcionamiento del sistema de extinción y control de incendios

para el proyecto de planta Riles de TRIO S.A. ubicado en Esquina Blanca 1117, Maipú.

**II.** **ALCANCE**

La descripción de este documento tiene como alcance el proyecto de planta RILES.

**IV. COMPONENTES DEL SISTEMA**

-Extintores de CO2

-Extintores de PQS

-Red Húmeda

-Alarma y detección automática de incendio

**V. DESCRIPCIÓN DEL SISTEMA**

El sistema de control de fuego contará con los siguientes tipos de extintores:

|Tipo de Extintor|Cantidad|Potencial de<br>Extinción|Capacidad en kilos|
|---|---|---|---|
|PQS|1|10 A - 60 B:C|10 kg|
|CO2|1|10 B:C|5 kg|

El sistema de red húmeda cuanta con el siguiente equipamiento:

|Tipo de Red|No Grifos|
|---|---|
|Húmeda|1|
|Manguera 1 1⁄2 pulg., 30 mts.|1|

El sistema de detección de incendio operativa en las instalaciones actuales contempla el
siguiente equipamiento:

|Equipamiento|Cantidad|
|---|---|
|Detectores de humo|4|
|Palanca|1|
|Alarmas|1|

Se adjunta plano descriptivo del sistema de control y extinción de incendios con el detalle
de la ubicación en la planta de cada uno de estos equipamientos anteriormente

señalados.
