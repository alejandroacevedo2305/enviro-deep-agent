Análisis de la capacidad de uso de suelo en

proyecto Nueva Aurora

# Metodología

Se utilizaron archivos en formato Shape File, de Información Geográfica asociada al Estudio
Agrológico de Suelos (CIREN). Además, en base a imágenes satelitales se elaboró un polígono del

área de estudio, correspondiente al área de emplazamiento del proyecto Nueva Aurora en

comuna de Ovalle provincia de Limarí.

Esta información se procesó en el Software de código libre Qgis 2.18 para interceptar la

información del Estudio Agrológico de Suelos con el polígono del área de estudio elaborado, luego

se calculó la proporción de las n posibles clases presentes en el área de estudio y finalmente se

procedió al análisis de resultados.

# Resultados y Análisis

El área de estudio presenta en toda su extensión solo una clase de capacidad de uso de suelo (Ver
Figura 1: Capacidades de uso de suelo en proyecto Nueva Aurora), la cual corresponde a la Clase

III. Estos suelos presentan moderadas limitaciones de uso y capacidad productiva regular, aún

cuando puede ser buena para algunos cultivos específicos. En cuanto a sus características físicas,

son suelos delgados de topografía plana o moderadamente inclinada (pendiente hasta un 5%).
