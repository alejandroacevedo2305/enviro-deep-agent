# Informe de Disposición

**Abril 2021**

La ceniza se recibió en forma normal durante el mes de abril. Se procedió a limpiarla y dejar la

escoria a un lado, la cual fue utilizada para mejorar caminos interiores en el sector El Peral.

La ceniza útil se utilizó para hacer pruebas con lodos sanitarios y aplicarlas a suelos de pastoreo de

tipo trumao arenoso, ubicado en la zona de Quilaco.

Recién se están viendo las proporciones de la mezcla y ver si surte el efecto deseado, utilizando la

ceniza como estructurante para el lodo.

Por una parte, se aplica la mezcla directa al suelo (100 ton) y en la otra se mantiene en pilas para

ver la generación de compost.
