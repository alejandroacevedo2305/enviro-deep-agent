# DECLARACIÓN DE IMPACTO AMBIENTAL LÍNEA DE FLUJO PAD INVERNADA ZG-A EMPRESA NACIONAL DEL PETRÓLEO INICIO DE EJECUCIÓN DEL PROYECTO

### Fecha: Mayo, 2021

## Inicio de Ejecución del Proyecto

Se tiene planificado iniciar la Fase de Construcción del Proyecto el 1 de diciembre de 2021,

sujeto a la obtención de la RCA y autorizaciones sectoriales respectivas. Se dará inicio a esta

fase con la implementación de la IIFF; mientras que se dará término con la instalación de

letreros informativos en el área, en febrero de 2022.

El presente documento, ha sido elaborado para ser subido al E-SEIA. El contenido de éste

se encuentra en el Capítulo 1 Antecedentes Generales y Descripción del Proyecto.
