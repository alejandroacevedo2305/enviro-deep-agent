**1.** **INTRODUCCIÓN**

Como parte de los estudios y antecedentes requeridos para la caracterización de los suelos

se ejecutó la exploración geotécnica mediante el muestreo de calicatas para determinar la

estratigrafía del sector.

Se realizó un total de 20 calicatas en el área de ejecución del Proyecto con el fin de recopilar

mayores antecedentes del área.

A continuación, se describen las calicatas desarrolladas en el Empréstito La Italiana (Detalle

en Anexo 17- Planta Calicatas).
