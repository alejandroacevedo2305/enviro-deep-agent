# White Ground Cover

DeWitt White Ground Cover is a
woven polypropylene fabric used
in groundcover applications where
light reflection is desired.
DeWitt White Groundcover is

formulated to resist uv
degradation, and allows water, air,
and nutrients to pass through the
fabric.
