**Alcance General del Proyecto**

 El proyecto Parque Fotovoltaico Andino Longotoma
suministrará energía al Sistema Eléctrico Nacional
para abastecer a hogares e industrias.

 - Este proyecto de Energías Renovables No
Convencionales se suma al camino que Chile está
siguiendo para generar energías limpias y libres de
contaminación.

 - Este es clave el de
parque para proceso
descarbonización con el que estamos comprometidos
como país y planeta para eliminar las centrales a
carbón al año 2050.

**Características del Proyecto**

**Capacidad de transmisión** : 230 MWp

**Hectáreas a utilizar** : el área de arriendo tiene
una superficie de 373,3 ha.

**Conexión** : Nueva SE La Ligua

**Almacenamiento** : 200 MW

**Suelo** : suelos que no cuentan con aptitud
agrícola.
