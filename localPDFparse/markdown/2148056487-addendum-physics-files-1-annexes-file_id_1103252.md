**Anexo No 8. Set Fotográfico Medio Humano Actualizado**

El presente documento viene a ser un complemento de las visitas a terreno realizadas para
el reconocimiento del Área de influencia del Medio Humano presentada en la DIA del

proyecto y en la ADENDA. A continuación, se adjunta un set fotográfico de los distintos

elementos encontrados al interior del predio y en el entorno del proyecto. Para mayor

información a modo de apéndice, se adjunta en formato .kmz, la ubicación exacta de cada

toma fotográfica.

**Ítem levantado fotográficamente:**

1. Proyecto (Interior del predio, Edificación, flora, etc.)

2. Infraestructura/ Áreas Verdes y Espacio Público (Transporte, Plazas, Parques,

etc.)

3. Equipamiento (Religiosos, Comunitario, Comercio, Salud, Seguridad.)

4. Vialidad del entorno (Uso vehicular y de estacionamiento de calles del

entorno del Proyecto.)
