# T555III
## TRENCHER

**SMARTTEC ELECTRONIC CONTROL SYSTEM.**

SmartTEC performance software helps optimize
productivity with real-time machine control
prompts. Monitor and record functions encourage
proactive machine maintenance and operator
performance analysis. Available on all Vermeer
TEC [®] Plus system-equipped machines.

**VANTAGE TRACK.** Relax in the confidence
that your investment is protected and your entire
closed-loop hydrostatic system is covered with
an optional 3-year/3000-hour Extended Care
Service program.

##### VERMEER.COM

**OPTIONAL AUTO TILT TRACK.** An oscillating
track frame allows the track to follow the ground’s
contour, providing a vertical trench on slopes up to
10 degrees.

**ENHANCED COOLING SYSTEM.** Improvements
to the cooling system on the T555III Commander [®]
3 trencher have resulted in a 5 percent increase in
available horsepower to the ditch.

**HYDROSTATIC ROCKWHEEL ATTACHMENT.**

Powered by a hydrostatic motor, the rockwheel
attachment has the ability to cut a trench up to
36" (91.4 cm) in depth and 6" (15.2 cm) in width.

**FULL-FUNCTION REMOTE CONTROL.**

The T555III is available with a wireless

remote control that can provide full operator
function remotely.
