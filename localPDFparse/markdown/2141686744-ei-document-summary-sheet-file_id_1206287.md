de situaciones y una pizarra que indique el horario de funcionamiento de trabajo, que de aviso en caso de
improvistos y se dejará permanentemente un número de teléfono para contacto directo con algún
encargado de obra. Este libro de denuncias estará disponible para la entidad fiscalizadora.
