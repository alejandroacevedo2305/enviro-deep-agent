# DPU 4045H Plancha vibratoria reversible, 40kN, 45cm/18in

**Código de artículo 0610051**

**Las placas de posibilidades ilimitadas para**
**todos los suelos**

Estas planchas vibratorias tienen las mejores características
para la compactación de diversos tipos de suelo,
especialmente los semicohesivos, que experimentan toda su
potencia. Las planchas destacan no sólo por su potencia sino
también por su velocidad de avance y retroceso variable, que
se puede ajustar hasta 28 m/min. Además, la técnica de
excitador de dos ejes proporciona la mejor profundidad de
compactación posible en prácticamente todos los suelos.

**Características y ventajas**

De aplicación flexible gracias a la velocidad de avance y retroceso regulable. Ideal
para la compactación de suelos en zanjas y grandes superficies, así como para el
vibrado de pavimentos adoquinados.

Anchos de trabajo variables gracias a placas adicionales incluidas en el contenido de
entrega.

Placas base de fundición de grafito modular (GJS-700) extremadamente resistentes al
desgaste.

Elevada seguridad por sistema de seguridad para marcha de retroceso. Este último
dispositivo evita que el usuario quede aprisionado entre la máquina y el obstáculo
durante los trabajos que se realizan marcha atrás.

Un robusto marco de protección y un punto de izaje central facilitan la carga y el
desplazamiento del equipo (por ejemplo, en zanjas).
