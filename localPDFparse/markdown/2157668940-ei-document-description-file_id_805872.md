|**1.2.2. Descripción breve del proyecto**<br> <br>El proyecto corresponde a una regularización de una ampliación realizada al proyecto original aprobado por<br>RCA N 224/2002, el cual consiste en la ampliación del muelle en 200 metros x 12 metros de ancho. Además,<br>se actualiza la modificación del proyecto ingresando la totalidad de las estructuras u obras anexas al recinto<br>portuario, las cuales se encuentran autorizadas sectorialmente, como así mismo se agregan actividades y/o<br>servicios de carga general y cabotaje actualmente en ejecusión.<br> <br>Es así como el presente documento gestionará la regularización de las siguientes estructuras e<br>instalaciones:<br> <br> <br>• Muelle: Ampliación en 2.400 m2 (largo 200m x 12m ancho).<br>• Galpón de 1.500 m2 (75m x 20m), la cual incluye las siguientes dependencias:<br>• Bodega: Instalación nueva de 900 m2 (45m x 20m).<br>• Oficinas: Instalación nueva de 200 m2 (10m x 20m).<br>• Comedor: Instalación nueva de 300 m2 (15m x 20m).<br>• Lavandería: Instalación nueva de 75 m2 (5m x 15m).<br>• Baños: Instalación nueva de 17,5 m2 (7m x 2,5m).<br>• Bomba de Combustible: Instalación nueva de 4 m2 (2m x 2m).<br>• 2 estanques de combustibles de capacidad de 49 m3.<br> <br>La relevancia de realizar estas ampliaciones se debe a que dada las características y requerimientos de la<br>región y la comuna, se requirieron instalar y ampliar ciertas instalaciones para poder abordar de manera<br>idónea las necesidades de Puerto Natales, potenciar la economía local y ayudar a la industria del sector.<br> <br>Debemos recordar que, dado a que la comuna de Puerto Natales se encuentra en la zona más austral del<br>país la manera de poder brindar apoyo en insumos y aportes es por la vía marítima. Además, considerando<br>la localización estratégica que posee el muelle, el cual está proyectado sobre el Seno de Última Esperanza,<br>el que presenta una diferencia de 1,5 m aproximado entre la marea alta y la baja; convirtiendo a esta zona<br>en un lugar adecuado para poder desarrollar esta actividad portuaria. Finalmente, la comuna de Puerto<br>Natales es una comuna relativamente aislada del resto del país comunicada por pocas vías de acceso desde<br>tierra.
