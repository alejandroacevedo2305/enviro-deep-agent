**1. INTRODUCCIÓN.**

atmosféricas, para las fases de construcción y operación del proyecto **“Parque**
**Solar Qanqiña”** el cual se ubicará en la comuna de Pozo Almonte, I Región de
Tarapaca.

Metropolitana".

Memoria de Cálculo de Estimación de Emisiones

**Ed** : Tasa de emisión diaria por tramo (g/día)
**e** : Factor de emisión de partículas por calle (g/vehículo-km)
**F** : Flujo vehicular (vehículos/día)
**L** : Longitud del tramo (km)
