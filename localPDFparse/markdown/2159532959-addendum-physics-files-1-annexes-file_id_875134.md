En el siguiente apéndice se presentan la memoria de cálculo entregado por el software de modelos

de propagación implementados para el proyecto evaluado.

##### 1. Configuración Software.

En la configuración del software encontramos, las componentes meteorológicas como temperatura,
humedad de aire, presión atmosférica, componentes asociados a atenuación acústica como la
absorción del aire, absorción del terreno, efecto de barreras, método de cálculo utilizados en la

modelación.

##### 2. Memoria de cálculo ESC1.- Construcción

2.1. Area Source (Fuentes de Ruido de área)
Las Area Source o Fuentes de Ruido de área representan fuentes emisoras de ruido incorporadas al modelo, la memoria de cálculo entrega el
nombre (Name), descripción de la fuente emisora (Desc.), ubicación geométrica mediante coordenadas UTM (X1, Y1), altura (Height), nivel de
emisión sonora por banda de octavas (LwTot 63 - LwTot 8k) y total (LwTot Tot) consideradas en cada fuente de emisión.
