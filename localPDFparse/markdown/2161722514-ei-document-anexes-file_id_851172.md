Inversiones e Inmobiliaria Plaenge Chile Ltda.

**1.** **Plan de seguimiento de las variables ambientales**

El Plan de Seguimiento de las Variables Ambientales se presenta como una herramienta

indispensable para garantizar la sostenibilidad y la gestión responsable del medio ambiente en

cualquier proyecto o actividad. El objetivo principal de este plan, bajo la supervisión de la

Superintendencia, es asegurar que las variables ambientales relevantes se monitoreen y

controlen de manera efectiva a lo largo de todas las fases del proyecto, considerando

especialmente el impacto del cambio climático.

Este documento establece directrices claras para la medición y control de los componentes

ambientales, incluyendo la identificación de puntos de control, la definición de parámetros de

evaluación, y la especificación de métodos de medición. Además, se establecen límites permitidos

o comprometidos, así como la frecuencia y duración del seguimiento ambiental.

Es fundamental destacar que el Plan de Seguimiento también contempla la adopción de medidas

y acciones de control necesarias para prevenir impactos ambientales significativos. En este

sentido, se promueve una gestión ambiental proactiva y responsable, orientada hacia la

conservación y mejora del entorno.

En resumen, este plan ofrece un enfoque integral para la gestión ambiental, proporcionando las

bases necesarias para evaluar, prevenir y mitigar los impactos ambientales derivados de cualquier

actividad humana, contribuyendo así a un desarrollo más sostenible y respetuoso con el medio

ambiente.

A continuación, se presenta el plan de acción derivado de las medidas y compromisos ambientales

adquiridos durante la presente DIA.

**Conclusión**

El Plan de Seguimiento de Variables Ambientales es esencial para garantizar la sostenibilidad y la

gestión ambiental responsable. Supervisado por la Superintendencia, su objetivo principal es

monitorear efectivamente las variables ambientales clave, incluyendo el impacto del cambio

climático, durante todas las etapas del proyecto. Proporciona directrices claras para la medición

y control ambiental, así como medidas preventivas para evitar impactos significativos. En

resumen, este plan ofrece un enfoque integral para evaluar, prevenir y mitigar los impactos

ambientales generados hacia la fauna, la contaminación acústica y atmosférica y así,

contribuyendo a un desarrollo más sostenible.
