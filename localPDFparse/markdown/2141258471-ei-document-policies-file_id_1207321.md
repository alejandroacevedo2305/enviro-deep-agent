## I.1. A NIVEL REGIONAL.

### I.1.1. Estrategia de Desarrollo Regional 2012 -2021

El 16 de noviembre de 2009 se firmó un Acuerdo Marco entre el gobierno Regional Metropolitano de Santiago (GORE RMS) y el
programa de Naciones Unidas para el Desarrollo, con el objeto de _“realizar acciones conjuntas de apoyo al desarrollo sustentable de la_
_Región Metropolitana de Santiago, el mejoramiento de las condiciones de vida de sus habitantes y el desarrollo de las capacidades de_
_su institucionalidad pública”._ En este marco de cooperación, el año 2011 se acordó desarrollar el Proyecto de construcción de la
Estrategia Regional de Desarrollo (ERD) para la Región Metropolitana de Santiago, para el periodo 2012 - 2021.

La Estrategia Regional de Desarrollo es el principal instrumento de planificación que poseen las regiones en Chile. En la Región
Metropolitana de Santiago, se ha promovido la instalación de un Sistema Regional de Planificación, que tiene como objetivo
transformarse en la carta de navegación que oriente a las instituciones públicas, empresas privadas y organizaciones ciudadanas para
construir una mejor región.

La Estrategia de Desarrollo Regional 2012 -2021 ha definido 5 Lineamientos Estratégicos Regionales (LER), los cuales se indican a
continuación:

1. Santiago - Región Integrada e inclusiva
2. Santiago - Región Equitativa y de Oportunidades
3. Santiago - Región Segura
4. Santiago - Región Limpia y Sustentable.
5. Santiago - Región Innovadora y Competitiva.

Cada uno de los lineamientos, se compone de objetivos estratégicos y objetivos operacionales. La relación que tiene el Proyecto con
