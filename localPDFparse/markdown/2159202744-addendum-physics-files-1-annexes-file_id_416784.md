## Grupo Generador J110K

El modelo J110K es un grupo electrógeno que entrega una potencia
standby de 110 KVA. Incorpora un motor marca John Deere (modelo
4045HF120), fabricado en USA. Su estanque de combustible tiene una
capacidad para 190 litros. Como todos nuestros equipos, posee un
excelente rendimiento y un consumo al 75% PRP de 16,5 L/h.
