# 1 ESPECIFICACION DE LA PLANTA DE TRATAMIENTO DE AGUAS SERVIDAS.

## 1.1 Introducción.

Se presenta la memoria de cálculo de cada etapa del proceso del sistema de tratamiento de aguas servidas,
tipo Lodos Activados, modalidad Aireación Convencional.

La planta de tratamiento de aguas servidas, similar al modelo **2TK-1 EQ** propuesto, cubrirá las necesidades
para las instalaciones de Foraction Planta la Unión.

El sistema de tratamiento de aguas servidas es de funcionamiento continuo, con una capacidad media diaria
total de 67,5 m [3] /día, generando agua con calidad para cumplir con el DS N° 90 "NORMA DE EMISIÓN PARA LA
REGULACIÓN DE CONTAMINANTES ASOCIADOS A LAS DESCARGAS DE RESIDUOS LÍQUIDOS A AGUAS
MARINAS Y CONTINENTALES SUPERFICIALES" (Tabla N°1 Límites máximos permitidos para la descarga de
residuos líquidos a cuerpos de aguas fluviales) y calidad de riego según lo establecido en NCh. 1.333
"REQUISITOS DE CLAIDAD DEL AGUA PARA DIFERENTES USOS).

El sistema de tratamiento de tratamiento diseñado consiste principalmente en las siguientes etapas de

tratamiento:

 - Planta de Tratamiento del tipo Lodos Activados modalidad Aireación Convencional modelo 2 TK-1 EQ,
considera las siguientes etapas:

 Etapa de sedimentación primaria, ecualización y elevación de las aguas servidas.

 Etapa de Aireación (Reactor biológico).

 Etapa de Sedimentación Secundaria de lodo.

 Etapa de Desinfección de efluente.

 - Sistema de Cloración (Tabletas).

 - Sistema de Decloración (Tabletas).

 Etapa de Digestión aeróbica de lodos.

 Etapa de acumulación de agua tratada.

## 1.2 Bases de Cálculos.

El sistema de tratamiento de aguas servidas está considerado para tratar las aguas servidas de las

instalaciones de Foraction Planta La Unión.

## 2.2 Etapa de ecualización y elevación de las aguas servidas

Para las variaciones horarias en el caudal de aguas servidas que se producen durante el día, como también los
cambios en la carga que esta tiene, se considera una etapa de ecualización y elevación de las aguas servidas
hasta la primera etapa de la planta de tratamiento, para así obtener un caudal constante y más homogéneo
en su composición al sistema de tratamiento.

que este es el punto donde llegarán todas las AA. SS generadas en las instalaciones.
