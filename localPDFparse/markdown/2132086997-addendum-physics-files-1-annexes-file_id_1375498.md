# ANEXO 7 Diagrama de Flujo y Balance de masa y energía

**ADENDA 1**
“Planta de Reciclaje de Aceites, Caucho y Plásticos Fuera de Uso”

**1** **BALANCE DE MASA**

Se consideran dos líneas de producción:

 - Para tratar neumáticos o plásticos. No se mezclan es uno u otro.

 - Para tratar aceites usados

Considerando condiciones normales (1 atm) y una temperatura entre 350 y 400 oC, para el
caso que la primera línea de producción sea alimentada con 25 [ton/día] de neumático o
plástico y la segunda línea de producción sea alimentada con 50 [ton/día] de aceites
usados, se tiene el siguiente balance:

|SUBPRODUCTO/PRODUCTO|CANTIDAD SEGÚN TIPO DE ALIMENTACIÓN<br>[ton/día]|Col3|Col4|
|---|---|---|---|
|**SUBPRODUCTO/PRODUCTO**|**NEUMATICO**|**PLASTICOS**|**ACEITE USADO**|
|Negro de Humo|8|-|-|
|Asfalto|-|4,25|3|
|Diesel|10|17,55|43,5|
|Chatarra|3,8|-|-|
|Gas Pirolítico|2|2|2|
|Agua con trazas|1,2|1,2|1,5|

**2** **BALANCE DE ENERGIA**

Considerando condiciones normales (1 atm) y una temperatura entre 350 y 400 oC, se tiene
un consumo de combustible (gas pirolitico) de 60 [kg/h] para el caso del neumático o
plástico, y de 160 [kg/h] para el caso del aceite usado.

Sólo cuando se haga partir la planta se necesitará GLP por un tiempo de 45 minutos. Esto
equivale a un consumo de 0,6 [kg/h]. Esto debería hacerse una sola vez al año, cuando se
revise el funcionamiento de los equipos y se pare la planta. Y esporádicamente para
situaciones de contingencia.
