# GENERADOR BASCULANTE MONOFÁSICO 10 kVA DIÉSEL

El grupo electrógeno “INMESOL” es una máquina de generación de energía eléctrica que se utiliza en aquellos
lugares donde no hay suministro de red o bien cuando se produzca un fallo de la red eléctrica.
El generador dispone de su marcado “CE”, y con cada una de ellas se emite la Declaración de Conformidad correspondiente
donde se especifica que esta cumple con el R.D 842/2002 Reglamento de Baja Tensión y con las Directivas Europeas.
Además cumple con la normativa chilena (SEC).
