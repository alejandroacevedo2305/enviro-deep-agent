**Establecimiento Inicio Ejecución del Proyecto**

Se estima el inicio de ejecución del proyecto será marzo del 2024, con la habilitación de la

instalación de faena. Estas instalaciones comprenderán principalmente oficinas para el

personal de la empresa constructora, bodegas, servicios higiénicos, y en general, lo que se

determine necesario para la buena ejecución de la obra.
