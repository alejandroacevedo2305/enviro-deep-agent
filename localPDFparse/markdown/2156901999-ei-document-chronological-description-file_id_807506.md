**A.7.2.** **Cronología**

La fecha estimada de inicio de la fase de operación correspondería al mes de **mayo 2025**, cuyo hito de inicio
será la recepción municipal del Edificio A (recepción parcial) y enero 2026 para el Edificio B. Se debe tener en
consideración que lo anterior está, además, condicionado a las fechas de aprobación de los distintos estudios
y autorizaciones que impliquen las obras. Esta fase está proyecta de forma indefinida, no contemplando una

fecha de término.
