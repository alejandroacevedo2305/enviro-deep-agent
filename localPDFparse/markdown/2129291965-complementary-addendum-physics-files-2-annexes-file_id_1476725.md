# Memoria de Cálculo Estimación de Emisiones Atmosféricas Proyecto “Curtiembre Agrícola Riñihue”

**2014**

Memoria de Cálculo de Estimación de Emisiones

**1.** **INTRODUCCIÓN.**

El presente Anexo contiene el detalle del cálculo de la estimación de emisiones
atmosféricas, para las fases de construcción y operación del proyecto
**“Curtiembre Agrícola Riñihue”** el cual se ubicará en La comuna de Maipú,
Región Metropolitana.

**2.** **FASE DE CONSTRUCCIÓN.**

La fase de construcción del proyecto _**“**_ **Curtiembre Agrícola Riñihue”** _,_ tendrá una
duración de 14 meses, de 10 meses para el montaje de maquinaria, con un total

de 24 meses.

El siguiente cronograma indica las actividades susceptibles de generar emisiones
durante la fase de construcción del proyecto:
