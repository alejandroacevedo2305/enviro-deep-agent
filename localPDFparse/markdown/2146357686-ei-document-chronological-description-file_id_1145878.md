**2.6.4. Mano de Obra Fase de Operación (Art. 19o a.6.4.)**

La operación del Proyecto, referida al funcionamiento y control de la seguridad del Parque
Solar Fotovoltaico Patagua no requiere de trabajadores, ya que la operación del Proyecto
se realizará de manera remota.

El personal de mantenimiento, corresponde a empresas contratistas externas, las cuales
realizarán las mantenciones de acuerdo al programa de operación del Proyecto.

**2.6.5. Descripción de las Actividades de Mantención (Art. 19o a.6.5.)**

La fase de operación involucra labores asociadas a labores de mantenimiento de los
equipos asociados a la operación. Estas actividades se describen a continuación.

**2.6.5.1.** **Mantenimiento**

El Mantenimiento considerado para el Proyecto incluye las siguientes sub-actividades:

 - Mantenimiento preventivo y limpieza.

 - Mantenimiento correctivo.

 - Mantenimiento por fallas.

- **Mantenimiento Preventivo y Limpieza**

El funcionamiento de un panel solar requiere niveles de mantención mínimos, asociados
principalmente a mantenerse limpios de polvo. La limpieza de los paneles se realiza
empleando agua sin ningún tipo de aditivo o detergente como base de dicha limpieza,
