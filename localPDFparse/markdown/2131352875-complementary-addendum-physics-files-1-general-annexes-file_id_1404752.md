**5.4.- Inflamabilidad**
En caso que la muestra contenga líquidos inflamables, el ensayo se realiza según método EPA 1010,
este método se basa en la determinación de la temperatura (°C) a la cual se inflama el residuo,
utilizando para este propósito el sistema de copa cerrada “Pensky Martens “. En el caso que la muestra
corresponda a un sólido el concepto de inflamabilidad corresponde a la velocidad de desplazamiento
de una llama, realizando el ensayo bajo determinadas condiciones experimentales (EPA 1030)
