**2.6.4. Mano de Obra Fase de Operación**

Durante la fase de operación, el Proyecto no requerirá mano de obra de forma permanente
al interior de las instalaciones. Para las labores de mantención y el lavado de paneles cuatro
(4) veces al año, se contratará a un tercero autorizado el cual realizará dichas labores en
función de la programación anual o requerimientos del parque.

**2.6.5. Descripción de las Actividades de Mantención**

Corresponde a la reparación de las instalaciones, tras fallas que comprometan la
transmisión de energía. Su envergadura depende de las anomalías producidas. Este
mantenimiento se realiza con programación de corto plazo, después de producida la falla y
generalmente involucra una estructura o un sector de la línea. Sin embargo, también se
consideran mantenciones programadas.

**2.6.5.1. Mantención**

El mantenimiento considerado para el Proyecto incluye las siguientes sub-actividades:

 - Mantenimiento preventivo.

 - Mantenimiento correctivo y por falla.

 - Actividades de limpieza.
