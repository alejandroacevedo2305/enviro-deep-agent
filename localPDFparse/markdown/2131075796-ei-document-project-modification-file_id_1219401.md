_**“ACONDICIONAMIENTO**_ _**DEL**_ _**HIDROCARBURO**_ _**GASEOSO**_ _**POZO**_ _**ACHE**_ _**ESTE**_ _**X-1”**_
1.3.8. MODIFICACIÓN A PROYECTO
_**1.3.8.**_ _**M**_ _**ODIFICACIÓN A**_ _**P**_ _**ROYECTO**_

El presente proyecto contempla incrementar la capacidad de acondicionamiento
de hidrocarburo gaseoso en Bloque Fell. El proyecto a modificar fue calificado bajo
la RCA 49/2015 y la Resolución Exenta 116/2015.

Por consecuencia, el proyecto podría incorporar un sistema de separación de
membranas o aminas (similar al que se encuentra operando).
