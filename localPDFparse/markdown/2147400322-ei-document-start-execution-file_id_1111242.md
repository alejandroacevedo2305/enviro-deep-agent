**2.8** **ESTABLECIMIENTO DEL INICIO DE EJECUCIÓN DE PROYECTO**

La obra que determina el inicio de la ejecución del Proyecto es la implementación de la instalación
de faenas, la que se estima para el primer trimestre del 2021, previa obtención de la aprobación
ambiental y demás permisos pertinentes.
