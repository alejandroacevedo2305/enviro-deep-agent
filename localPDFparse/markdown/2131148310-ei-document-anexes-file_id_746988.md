# MEMORIA

**1.** **Generalidades**

El presente proyecto se refiere a la construcción de la Red de Alcantarillado de Aguas Lluvias
para el Condominio Kennedy 866, formado por 320 viviendas acogidas al D.F.L. N° 2, que se
construirá en la comuna de Rancagua.

El terreno en que se construirán las viviendas tiene una superficie de 2,00 ha. y sus límites
son los siguientes:

Norte : Calle Los Alpes.

Sur : Comisaria.

Oriente : Avenida Kennedy.

Poniente : Otros propietarios.

**2.** **Calidad de Suelos**

De acuerdo al estudio, ejecutado por Geocav ltda, el terreno presenta la siguiente estratigrafía
típica:

Horizonte H-1: Capa vegetal de 0,30 m de espesor, presenta matriz arcilla limosa de color
café claro y humedad media. Se aprecian raíces y raicillas.

Horizonte H-2: Arcilla limosa de color café claro, humedad media, consistencia media,
estructura migajón, se aprecian raíces y raicillas.
Clasificación CL-ML, según el sistema U.S.C.S.
Espesor del estrato 0,90 m.

Horizonte H-3: Arcilla limosa de color café, humedad media, consistencia media a alta y
estructura migajón.
Clasificación probable CL, según el sistema U.S.C.S.

Nivel freático: No se detecta a la profundidad explorada (01/2012).

**3.** **Solución General**

Las aguas lluvias serán infiltradas en el terreno mediante un sistema de sumideros y zanjas de
infiltración, diseñado de acuerdo con las recomendaciones de la publicación “Técnicas
Alternativas para Soluciones de Aguas Lluvias en Sectores Urbanos. Guía de Diseño”, del
Ministerio de Vivienda y Urbanismo.

# 4.6 Cálculo de los Colectores

La capacidad de los colectores se ha calculado para H/D ≤ 0,8. Por lo tanto, para cada tramo
de los colectores se verificó que el caudal de escurrimiento fuera menor que la capacidad para
una altura normal del 80% del diámetro. Se adoptó este valor por razones de seguridad, aún
cuando la capacidad máxima de un colector se tiene con una altura del 94% del diámetro del
tubo.

**6.** **Cálculo de Sumideros**

Para determinar el número de sumideros a proyectar, se procedió a definir su ubicación de
acuerdo con las áreas tributarias definidas. Luego, se determinaron los caudales hidrológicos
afluentes a cada sumidero, los que deberán ser menores a la capacidad de los sumideros. El
ancho de cuneta inundada está entre 0,79m y 1,22m, con un promedio de 1,04m.
