# boomer m2 c

## Equipado con perforadoras COP 1638, COP 1838 o COP 2238

Equipo hidráulico para minería y túneles de dos brazos con un avanzado sistema de
control del equipo para una alta precisión y productividad. Disponible con una variedad
de perforadoras de roca para adaptarse a los requisitos del cliente.

**Características estándar**

**"Perforadora** **"Sistema de control**

 - Perforadora de roca COP 1638, COP 1838 o COP 2238 con sistema de

doble amortiguación para una duración óptima de los consumibles

 - Carcasa presurizada y superficies de contacto para reducir la

contaminación interna de la perforadora

 - Sistema de control de plataforma avanzado (RCS) que permite varios niveles de

automatización para adaptarse a los requisitos del cliente

 - Las funciones estándar incluyen colocación suave de collares y la función

antiatascos Alimentación controlada por presión de rotación (RPCF)

 - Sistema integrado de diagnóstico y registro de eventos para ayudar en el

**»Auge hidráulico** mantenimiento del equipo

 - Panel de operador interactivo con pantalla a todo color

mantenimiento del equipo

 - Pluma de servicio pesado BUT 35 SL con suspensión de trípode doble para

una sujeción paralela hidráulica precisa en todas las direcciones

**"Transportador**

**»Alimentación hidráulica** - Portaequipajes robusto y articulado con tracción en las cuatro ruedas

 - Alimentación de aluminio de alta resistencia de la serie BMH 6000 con

alta resistencia a la flexión y torsión para una máxima durabilidad

 - Manguitos de acero inoxidable a presión y contactos de polímero

 - Propulsado por un motor diésel de 4 cilindros aprobado por EPA III/COM

III (Tier 3/Stage IIIA)

 - Cuatro conectores para una configuración estable

**"General**

 - Techo protector telescópico aprobado por FOPS

 - Carrete de cable

 - Compresor accionado hidráulicamente

 - Bomba de refuerzo de agua accionada hidráulicamente

 - Luces de trabajo
