elección de la normativa mencionada debe considerar y/o justificar técnicamente este criterio, ya

aplicación elegida por el titular no representa el peor escenario, por lo que corresponde aplicar la
normativa de referencia para zonas residenciales con proyecto existente (PTAS) cuyo límite de olor
