# Memoria de Cálculo Capacidad Pretratamiento Proyecto de Ampliación PTAS Rancagua, VI Región

## Elaborado por: Octubre 2018

**1. Objetivos**

La presente memoria provee apoyo técnico para la confección de una memoria de
cálculo del pretratamiento que dé respuesta a consultas realizadas por la
Superintendencia de Servicios Sanitarios (SISS). En particular:

_Se solicita justificar técnicamente, mediante una memoria de cálculo, que la_
_capacidad de las rejas finas, desarenadores y desgrasadores es suficiente para_
_absorber el aumento de carga y caudal hasta el año 2030._

**2. Capacidad rejas**

Se verificó el funcionamiento de las rejas medias en condiciones futuras, esto es para
los caudales promedio y punta horario proyectados al 2030 (1101 L/s y 1788 L/s,
respectivamente).

Las mismas verifican holgadamente para el caudal promedio y sin mayores
inconvenientes para el caudal punta horario futuro, salvo en el parámetro velocidad de
paso a reja atascada que se encuentra en el límite superior pero igualmente operativo.
Para el caudal punta horaria del año 2030, para sacar una reja fuera de operación
para mantenimiento se precisará utilizar el tercer canal existente. Allí se podrá operar
con la reja manual existente en la actualidad (dado que su uso es temporario).

En resumen, las actuales rejas medias tienen capacidad suficiente para absorber el
caudal punta horario futuro y en consecuencia no se requieren modificaciones.

En resumen, los actuales desarenadores/desengrasadores en operación 1+1 tienen
capacidad suficiente para remover arenas de un corte mayor o igual a 0,16mm de
diámetro, para el caudal medio y punta horario futuro y en consecuencia no se
requieren modificaciones de estas unidades.

Cabe señalar para los caudales medios y punta del año 2030, en un evento de sacar
una desarenador de operación para mantenimiento se precisará, se tendrá una menor
eficiencia de retención de arenas, condición a la cual pasara un mayor contenido de
arenas a las unidades posteriores al desarenado.
