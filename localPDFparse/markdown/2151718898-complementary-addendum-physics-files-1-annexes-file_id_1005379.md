#### I. Descripción de proyecto

1. En relación a la descarga de emergencia de la PEAS Cabecera, ilustrada en la figura I-1 de la Adenda de

la DIA “Configuración Sistema de Pretratamiento y PEAS”, el titular indica que la capacidad de descarga a

orilla de playa es de 1.844 l/s. Al respecto, se solicita describir y respaldar dicha capacidad.

**Respuesta:**

Para calcular y respaldar el caudal de porteo de la descarga de emergencia, que asciende a 1.844 L/s, se

utilizó el método de Manning, para ello, el cálculo del caudal de porteo considera lo siguiente:

1. Con el Diámetro y la Altura de Agua de los Datos de Entrada se calculó el Angulo
2. Con el Diámetro y el Ángulo se calculó el Área Mojada
3. Con el Diámetro y el Ángulo se calculó el Perímetro Mojado
4. Con el Perímetro Mojado y Área Mojada se calculó el Radio Hidráulico

5. Con la Pendiente, Coeficiente de Rugosidad y el Radio Hidráulico se calculó la Velocidad.

6. Con el área Mojada y la Velocidad se calculó el Caudal de Porteo

Por otra parte, la metodología desarrollada considera las siguientes variables:

1. Diámetro de la tubería

2. Capacidad para Transporte Gravitacional (95% de tubería llena)

3. Pendiente (se consideraron las pendientes señaladas o calculadas de los planos).

4. Coeficiente de Rugosidad de Manning

A fin de realizar el cálculo se consideraron dos escenarios, a saber, el primero considerando el coeficiente

de Manning normalmente adoptado para las características de las tuberías que corresponde a n= 0,011.

El segundo escenario considera una sensibilización con coeficiente de Manning más desfavorable

(n=0,013).

Asimismo, se analizaron las pendientes corresponden a 3 valores en el colector Chalinga (Longitud

Completa de Planos, Longitud de Inicio a Empalme de Planos y Pendiente señalada en Planos) y 2 en el

colector Avenida Argentina (Longitudes de Empalme a C2 y de C2 a CE). Se evaluaron todas las pendientes

de cada colector, que corresponden a las siguientes:

1. 0,017, 0,020 y 0,013 m/m para el colector Chalinga.

2. 0,046 y 0,017 m/m en el colector Avenida Argentina

A partir de las pendientes se obtuvieron los consecuentes caudales asociados:

1. Caudal de porteo de 2.569 y 2.179 l/s para los colectores Chalinga y Avenida Argentina con n

= 0,011

2. Caudal de porteo de 2.174 y 1.844 l/s para los colectores Chalinga y Avenida Argentina con n

= 0,013
