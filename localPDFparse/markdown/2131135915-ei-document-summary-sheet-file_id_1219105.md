**D.** **COMPROMISOS AMBIENTALES VOLUNTARIOS**

Los titulares se comprometen a establecer un plan de comunicación y manejo con las

comunidades aledañas (organizaciones sociales con personalidad jurídicas) al lugar de

emplazamiento del Proyecto. Para ello, se mantendrá una pizarra informativa con los plazos de las

obras y horarios de faenas ruidosas. Se dispondrá, también, de un encargado de recibir y buscar

solución a posibles quejas de la comunidad disponiendo los medios de comunicación pertinentes

para estos temas.
