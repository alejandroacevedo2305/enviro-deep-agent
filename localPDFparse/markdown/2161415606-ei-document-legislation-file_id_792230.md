## BODEGA GEMAPRO MODELO BRP-3 PANEL Almacenamiento temporal de Residuos Peligrosos MEMORIA DESCRIPTIVA DE MATERIALES DE CONSTRUCCION

**1.** **INTRODUCCIÓN**

Este documento incluye una descripción de los materiales constructivos utilizados para el
diseño y fabricación de una bodega para el almacenamiento de Residuos Peligrosos (Modelo
BRP-3), con capacidad de 12 tambores.

**2.** **DIMENSIONES DE LA BODEGA**

|Frente (mm)|3,05|
|---|---|
|Fondo (mm)|2,3|
|Alto (mm)|2.800|
|Peso (Kgs)|1.700|

|---|---|
