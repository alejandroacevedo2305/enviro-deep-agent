# 1.7 Establecimiento del Inicio de Ejecución del Proyecto

proyecto, de modo sistemático y permanente, es el replanteo topográfico y labores previas al

inicio de la construcción.

# 1.8 Información de Negociaciones con Interesados

realizado negociaciones con interesados.
