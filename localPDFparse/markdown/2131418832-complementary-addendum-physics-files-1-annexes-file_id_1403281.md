Sin embargo, nos comprometemos como empresa si se efectuarse un hallazgo
arqueológico o paleontológico durante las excavaciones del proyecto se paralizara la obra

de Monumentos Nacionales, para que este organismo determine los procedimientos a

seguir, cuya implementación será a nuestro cargo.
