**2.10** **Establecimiento del inicio de ejecución de proyecto**

El inicio del proyecto es la reubicación de la planta de RILES a realizarse durante el mes
de julio de 2019, una vez que se obtengan las autorizaciones ambientales y sectoriales
correspondientes.
