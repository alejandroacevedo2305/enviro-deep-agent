# FICHA TÉCNICA

**BODEGA RF120 10 M2 PUERTA LATERAL 2 ACCESOS**
**CÓDIGO : RF120-10M2 L2A 2PD**

**DESCRIPCIÓN**

Bodega especialmente diseña para incompatibilidad de almacenamiento
según DS43 Art17, La bodega consta de muros divisorios y sistema de
contención independiente que permite almacenar residuos o sustancias
peligrosas incompatibles entre sí en una misma bodega.
