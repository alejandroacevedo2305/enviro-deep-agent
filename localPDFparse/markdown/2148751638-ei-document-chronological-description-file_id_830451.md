# 1.3.8 Modificación de proyectos

La Planta Yerbas Buenas de EISA tiene como objetivo suministrar el vapor de proceso a la Planta CMPC

Cartulinas.

La Planta cuenta con las siguientes Resoluciones de Calificación Ambiental:

 - Resolución Exenta N° 096/2001, califica ambientalmente favorable el proyecto “Calderas

Industriales Yerbas Buenas”.

 - Resolución Exenta N° 60/2006, califica ambientalmente favorable el proyecto “Caldera a Biomasa

en Maule”.
