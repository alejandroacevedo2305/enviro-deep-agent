||El objetivo del Plan de Gestión de Olores (PGO) para el Proyecto "Sistema<br>de Tratamiento de Riles Viña Inverco" es establecer un marco integral para<br>la gestión efectiva de las emisiones de olor generadas por el sistema de<br>tratamiento de riles. El PGO tiene como propósito principal la prevención,<br>reducción y control de los olores molestos asociados con las operaciones<br>del proyecto, asegurando el bienestar y la calidad de vida de los habitantes<br>y el medio ambiente circundante.<br> <br>**Objetivos Específicos:**<br> <br>1. Realizar un monitoreo constante de las emisiones de olor para detectar<br>cualquier incremento en la concentración de olores y tomar acciones<br>correctivas oportunas en caso de superar los límites establecidos.<br>3. Fomentar la participación y el diálogo con la comunidad local,<br>proporcionando información transparente sobre las acciones del PGO y sus<br>resultados, y promoviendo la adopción de buenas prácticas para prevenir la<br>generación de olores ofensivos.<br> <br>**Descripción:**<br> <br>En el marco del proceso de calificación ambiental del "Sistema de<br>Tratamiento de Riles Viña Inverco", el titular del proyecto presentará este<br>documento detallado que tiene como objetivo describir las acciones de<br>seguimiento y control que se implementarán en el proyecto para evitar la<br>generación de olores molestos en las áreas cercanas a las operaciones del<br>proyecto que puedan afectar a las comunidades cercanas.<br> <br>**Justificación:**<br> <br>El presente PGO se presenta para describir las acciones de seguimiento y<br>control que eviten la generación de olores molestos en las áreas cercanas.|
