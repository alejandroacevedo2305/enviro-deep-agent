# 5.5 ESTABLECIMIENTO DEL INICIO DE EJECUCION DEL PROYECTO

El acto mínimo que dio inicio de la ejecución de modo sistemático y permanente
al proyecto fue el reubicación de fondeos y recambio primera línea madre de
100m por una de 200m (Long-line) para el cultivo hidrobiológico en el área de
concesión para el aumento de biomasa.
