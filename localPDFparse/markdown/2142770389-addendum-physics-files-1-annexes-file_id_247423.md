# RECOLECTOR ORV-419 19m [3]

**®**

#### Descripción

Recolector de basura estanca de carga trasera modelo ORV 420 para el uso intensivo de circuitos de
recolección urbana, interurbana e industrial. Especial para municipios y empresas contratistas. Baja
emisión de ruidos inferior a 90 dB en el proceso de recolección, lo hace ideal para el funcionamiento
nocturno. Incluye alza contenedor, para contenedores de 120, 240, 360, 770 y 1.100 litros; pudiendo
levantar contenedores (Multi-norma).
Nivel de compactación 1:3 a 1:7 dependiendo del tipo de basura a compactar, capacidad de carga
sobre 10 TON.

#### Incluye

Controles de accionamiento ciclo de compactación trasero. Parada de emergencia para detención de
las funciones en caso de accidente. Ubicado en ambos costados de la cola.
Bajo nivel de ruido 78 dB. Insonorización de la cola, con material fonoabsorbente en laterales, fondo y
techo.
Seguro automático entre la cola y tolva, con accionamiento hidráulico.
Estanque para líquidos percolados 160 Lts.
Pisaderas plegables.
Sello de goma entre caja y cola, estanco.
Baliza.

Pintura poliuretano.
Aceleración automática.
