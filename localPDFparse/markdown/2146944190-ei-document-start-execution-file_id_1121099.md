# WIND 3 SpA Establecimiento del Inicio de Ejecución del Proyecto

De acuerdo a lo solicitado en el Artículo 16 del DS 40/12 MMA, el hito de inicio de la ejecución del

proyecto, de modo sistemático y permanente, corresponde a:

 - Preparación de suelos para la instalación de faenas durante abril de 2022.
