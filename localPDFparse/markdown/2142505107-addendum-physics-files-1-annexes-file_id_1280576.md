C = C c1 ∗M 1 + C c2 ∗M 2 + ⋯... ... ... ... . . C cn ∗M n

M1.....M2= Masas de los materiales combustibles integrantes, de calores de combustión Cc1..... Ccn
respectivamente, expresadas en kg.

Cálculo de la densidad de carga combustible media:

D c = [C] S

En donde:

Dc= Densidad de carga combustible media del edificio o parte de él, expresada en

MJ/m2 o Mcal/m2;

C= Carga combustible

S= Superficie de planta correspondiente (del edificio o parte de él), expresada en m2.
