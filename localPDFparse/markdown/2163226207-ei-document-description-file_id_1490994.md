# SISTEMA DE DRENADO DE AGUAS

OBRA : Sistema de Infiltración Chilemink.

UBICACION : Planta Chilemink Mostazal

PROPIETARIO : Criaderos Chilemink Ltda.

# MEMORIA

MÉTODO CIENTÍFICO DE DIMENSIONAMIENTO DE DRENAJE

CALCULO ZANJA DRENANTE

Ancho (A) : 4 m

Largo (L) : 50 m.

Alto (H) : 5,5 m. (Altura granular dren)

Superficie de fondo = L x A = 200 m [2]

Coeficiente Absorción Fondo Suelo (K 5 ) : 145 L/m [2] *día.

Total Capacidad absorción fondo = 29.000 L/día.

CALCULO CAPACIDAD TOTAL DE ABSORCIÓN DISEÑADA

La capacidad total de absorción diseñada corresponde a suma total zanjas drenantes.

Capacidad total de absorción de sistema:

C.T. = 3 pozos *29.000 L/día= 87.000 L/día (87 m [3] /día)
