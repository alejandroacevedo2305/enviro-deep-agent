# COMPARACIÓN DE TECNOLOGÍAS SISTEMA DE TRATAMIENTO AGUAS SERVIDAS AIREACIÓN CONVENCIONAL VS BIODISCOS AQUAVANT

## COMPARACIÓN SISTEMA DE TRATAMIENTO AGUAS SERVIDAS CONVENCIONAL VS BIODISCO AQUAVANT

|ITEM|SISTEMA CONVENCIONAL AIREACIÓN|BIODISCO AQUAVANT|
|---|---|---|
|**CAPEX Y OPEX**|Menor CAPEX - Mayor OPEX|Mayor CAPEX - Menor OPEX|
|**Consumo eléctrico**|Consumo eléctrico requerido por sistema de aireación,<br>compresores, impulsión y control.|Sólo<br>requiere<br>de<br>consumo<br>el<br>motorreductor y control.<br>Consumo 5 a 10 veces menor en<br>comparación<br>a <br>un<br>sistema<br>convencional.|
|**Ruidos**|Ruidos por compresor y sopladores.<br>En algunos casos, se necesita obra civil insonorizada|Bajo en ruidos ya que el sistema tiene<br>su<br>tapa<br>de<br>protección<br>y <br>motorreductor no genera ruidos<br>significativos.|
|**Olores**|Puede generar mal olor si es que el sistema de aireación<br>es deficiente.|Sin olores|
|**Flexibilidad caudal**<br>**de tratamiento**|Sin flexibilidad. Sistema diseñado a medida.<br>Requerimiento del 40 % de agua servida de la capacidad<br>de diseño como mínimo para operar.|Muy<br>flexible,<br>hasta<br>200%<br>de<br>sobrecarga<br>por<br>un<br>periodo<br>determinado.<br>No hay requerimientos mínimos.|
|**Control**|Necesita control de acuerdo con cantidad de lodo, carga<br>de entrada.|Completamente automática. Lodos<br>son recirculados al primer estanque.|
|**Obras civiles**|Requiere obras civiles complementaria y de mayor<br>envergadura<br>(estanque<br>biológico,<br>estanque<br>de<br>sedimentación)|Solo<br>requiere<br>radier<br>de<br>emplazamiento para el tratamiento<br>biológico y desbaste primario. El<br>decantador lamelar viene incluido en<br>los sistemas de biodiscos.|
|**Lodos generados**|Retención de lodo activado 2 a 3 días.<br>La disposición de lodo debe ser en un lugar autorizado.|Retención de lodo de 2 a 3 meses.<br>Genera un lodo inactivo que se puede<br>disponer de forma regular.|
|**Mediciones**<br>**requeridas**|Requiere de sensor de oxígeno para control.|No requiere control permanente.|
|**Operación**|Necesita operador de forma permanente.|Sólo necesita 1⁄2 hora por día para<br>verificar que la planta se encuentra en<br>funcionamiento.|
|**Mantenimiento**|Mantenimiento general + sopladores + parrilla de<br>difusores entre otros|Mantenimiento<br>general<br>+ <br>mantenimiento<br>preventivo<br>a <br>motoreductor (1 vez al año).|
