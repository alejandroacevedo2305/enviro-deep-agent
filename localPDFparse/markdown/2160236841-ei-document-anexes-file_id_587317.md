El muestreó desarrólladó córrespónde a un muestreó sistema ticó, el cual facilita la lócalizació n

córrespóndiendó lós estratós a lós tipós de ambiente de fauna definidós previamente, y dirigidó,
en dónde el especialista utiliza criterió de expertó para lócalizar de manera efectiva el sitió de
muestreó pór taxó n y/ó tipó de muestreó.
