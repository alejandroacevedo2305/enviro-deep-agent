**Cronograma de la fase de operación**

La fase de operación se iniciará al momento en que las primeras viviendas sean entregadas

a sus propietarios y cuyas actividades se relacionan al uso de las viviendas por parte de las

familias.

Para dar inicio a la etapa de operación es necesario que las viviendas estén completamente

habilitadas para su uso, habiendo concluido completamente la fase de construcción, por lo

que se espera que el inicio de la fase de operación del proyecto sea el año 2019, año en que

se entrega la primera etapa del proyecto.
