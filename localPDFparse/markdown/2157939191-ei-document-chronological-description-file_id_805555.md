**Declaración de Impacto Ambiental “Planta**

**Fotovoltaica Cahuil Solar II”**

Descripción de Proyecto

**1.2.9 Información de Negociaciones**

Al respecto, se informa que no han existido negociaciones con interesados de las cuales se

hayan desprendido medidas de compensación o mitigación ambiental. Se indica,

igualmente que, si durante el proceso de evaluación se presentaran negociaciones, de las

cuales se desprendan medidas de mitigación ambiental o compensación, éstas serán

oportunamente informadas a la autoridad, de manera que sean incorporadas a los registros

del proceso de evaluación ambiental.

**1.2.10 Etapas del Proyecto**

Según lo prescrito por el artículo 14 del Reglamento del SEIA, el Proponente deberá

informar si el proyecto o actividad sometido a evaluación ambiental, será desarrollado por

etapas, caso en el cual deberá indicar cada una de ellas, el objetivo y las razones o

circunstancias de que dependen la ejecución de las mismas. Asimismo, deberá informar las

obras y acciones pertenecientes a cada una de ellas y su duración estimada.

Al respecto, se informa que el presente Proyecto no se ejecutará por etapas.
