**Método Gráfico de Estabilidad**

Los análisis se realizan mediante el Método Gráfico de Estabilidad, el cual fue introducido
por **Mathews (1980)** y más tarde modificado por **Potvin (1989)** y **Nickson (1992)**, en base a
casos históricos recopilados de minas subterráneas canadienses, para la determinación
empírica de estabilidad de excavaciones.
El Método Gráfico depende de dos parámetros, el Número Modificado de Estabilidad (N’) y
el Radio Hidráulico (HR).

El N’ se calcula según el siguiente formulismo:

N ′= Q ′ ∗A∗B∗C

Dónde: Q’: Es el Índice de Calidad de Túneles Modificado de **Barton et al** . **(1974)** ; A: Factor
de condición de esfuerzos; B: Factor de ajuste por orientación de estructuras y C: Factor de
ajuste por efecto de gravedad.

El Radio Hidráulico se obtiene a partir de las dimensiones del caserón o cámara de
explotación, calculando el área de la zona a evaluar y su perímetro, mediante la siguiente

formula:

HR =Á rea / Per í metro

Para el análisis se evalúa el Radio Hidráulico obtenido según las dimensiones de cada
cámara de explotación, con el Radio Hidráulico Admisible. Éste último parámetro es
obtenido a partir del valor N’, de acuerdo con la fórmula de **Villaescusa (2007** ), entregada

a continuación.

HR ADM = 10 [⌊0,5730 + 0,338 log(N] [′] [)⌋]
