## BATTERY ENERGY STORAGE SYSTEM (BESS)

Los componentes internos del sistema de almacenamiento BESS están diseñados, ensamblados en

fábrica y probado para evitar fallas. Cada contenedor está equipado con un amplio monitoreo y

sensores redundantes que informan de forma inalámbrica al Titular del Proyecto para respuestas de
seguridad automatizadas.

La supervisión remota es un órgano de vigilancia y ayuda a abordar de manera proactiva los eventos
antes de que surjan emergencias.

A continuación, se especifican los elementos del sistema de almacenamiento en cuanto a las

## Supresión de incendios

Las baterías BESS cuentan con un sistema de supresión de incendios. Esto utiliza un rociador seco
que se instala de fábrica.
La imagen a continuación se ilustran los cuatro puertos de rociadores secos en cada extremo del
contenedor:

**Figura 2. Puertos de rociadores secos**

Fuente: Información referencial de un proveedor, 2024.

El rociador interno se activa automáticamente en presencia de fuego y/o calor excesivo. Esto utiliza

una válvula mecánica expuesta externamente en cada cabezal de aspersor dentro del contenedor.

## Sistema de refrigeración

El contenedor viene ensamblado en fábrica con el sistema de refrigeración. Este sistema ya está
cargado en las líneas del enfriador y en el tanque de expansión a nivel para adaptarse a la puesta en
servicio inicial. El enfriador se prueba en fábrica para garantizar un funcionamiento de producción
confiable.
Además, se incluye un sistema enfriador para respaldar el enfriamiento de las baterías y los entornos
del gabinete. El sistema de refrigeración instalada de fábrica requiere un mantenimiento estándar
para garantizar el funcionamiento adecuado a lo largo del tiempo. El enfriador es un sistema crítico
que se utiliza para enfriar las placas frías del compartimiento de la batería que controlan la

temperatura.

## Sistema deshumidificador

El sistema deshumidificador viene instalado de fábrica en cada compartimiento de batería. Este
sistema crítico protege los componentes electrónicos y el interior del gabinete contra la oxidación.

El sistema de almacenamiento de energía BESS se entrega completamente integrado y probado en
fábrica, incluyendo todas las pruebas necesarias para asegurar su correcto funcionamiento.
Además, está equipado con sistemas de detección de incendios y un sistema de refrigeración que
regula las altas temperaturas internas, garantizando su capacidad para gestionar emergencias de

forma autónoma.

El Titular también cuenta con un sistema de monitoreo continuo tanto del interior como del exterior

del contenedor. En caso de detectarse un aumento descontrolado de la temperatura, el sistema se
desconectará de manera inmediata para evitar cualquier eventualidad además de dar aviso
inmediato al cuerpo de bomberos de la comuna.
