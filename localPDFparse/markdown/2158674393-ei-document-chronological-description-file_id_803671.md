# 1.1.1 Cronología General del proyecto

La ejecución de las obras ocurrió en los años 2009, 2010, 2016 y 2019; donde se realizaron distintas
actividades o acciones para la construcción, funcionamiento y productividad de la planta de
tratamiento de RILES, junto a la planta productiva y bodegas.

Describir las principales partes que conformaron la fase construcción y fase operación de la

instalación de sistema de tratamiento de riles.
