**Inicio de ejecución de las obras del proyecto**

Las labores que marcan el inicio del proyecto (Etapa de Construcción) corresponden a las
establecidas en el numeral 1.6.1, denominadas como Liberación de Áreas, las cuales

contemplan:

 - Replanteo de las áreas de trabajo

 - Rescate de especies en categoría de conservación

 - Retiro y acopio de material de escarpe
