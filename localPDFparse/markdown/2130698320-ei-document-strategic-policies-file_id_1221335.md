**8. Relación del proyecto con los planes evaluados estratégicamente.**

No existen para el área del proyecto planes, políticas o programas evaluados
estratégicamente.

En cuanto a los Planes Reguladores Urbanos: El área del proyecto se ubica, en una zona
rural, no regulada, por lo tanto no le son aplicables los planes Reguladores Urbanos o
Interurbanos. Tampoco se encuentra en algún área de extensión de alguno de los planes
antes señalados.
