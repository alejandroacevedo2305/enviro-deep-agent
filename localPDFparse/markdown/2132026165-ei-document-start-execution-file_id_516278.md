**1.1.1** **Inicio de ejecución del Proyecto**

De acuerdo a lo señalado en el artículo 16 del Reglamento del SEIA, se informa que la fecha

estimada de inicio del Proyecto es el mes de Junio del año 2017, lo cual estará sujeto a la

obtención de la calificación ambiental favorable.

El acto que establecerá el inicio del Proyecto es la movilización de los equipos para llevar a cabo la

primera ejecución de prospección.
