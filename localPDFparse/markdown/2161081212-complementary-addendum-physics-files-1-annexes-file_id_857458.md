# Informe Índices de Absorción Proyecto Parque Fotovoltaico Acuyo Región de Valparaíso, Chile.

**1.** **ANTECEDENTES GENERALES**

Con motivo de la construcción del proyecto de construcción del parque fotovoltaico
Acuyo, se ha encargado la elaboración del presente informe, con el fin de
establecer el índice de absorción del terreno, para justificar dimensionamiento de
sistema de infiltración en base a drenes de infiltración para la etapa de
Construcción y operación, consisten en los sistemas para Planta de tratamientos y
Fosa séptica respectivamente.

La elaboración del ensayo, se efectuó basado en los antecedentes descritos en el
D.S. No 236/26 Reglamento General de Alcantarillados Particulares, del Ministerio
de Salud.

Finalmente, con este documento, será posible generar las bases de diseño para los
sistemas de infiltración descargado por el sistema de tratamiento de aguas servidas
