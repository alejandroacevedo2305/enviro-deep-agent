# 3.5.2. Cronología

El proyecto se desarrollará por un período aproximado de 18 meses, en los cuales se

ejecutarán las obras constructivas de la Etapa 1, hasta la recepción de las viviendas. Según

lo indicado anteriormente, la vida útil del proyecto se considera indefinida. Bajo este

contexto, se expone la programación de las actividades que contempla esta fase. En la

Tabla 8, se presenta el cronograma general de las actividades asociadas a la construcción

del proyecto.

Por otro lado, como aún no se han definido las características de la segunda Etapa del

proyecto el cronograma se limita únicamente a especificar los obras y acciones de la

primera Etapa.
