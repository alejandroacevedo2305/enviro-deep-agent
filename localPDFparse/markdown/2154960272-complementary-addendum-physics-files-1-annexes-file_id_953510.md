La STR-138, es una compacta y muy completa
máquina de perforación para realizar, tanto pruebas
de suelo SPT, como perforación rotatoria y saca
núcleos con pescador.

Este equipo de perforación de suelos, es perfecto
para trabajos a profundidades máximas de entre
45 y 75 metros. Cuenta con mástil retráctil,
martillo de SPT manual o automático, prensa
de tubería, una poderosa cabeza de perforación
de cuatro velocidades y bomba de lodo o agua
para perforaciones con wet swivel. Una pequeña
perforadora de suelos con gran poder, a un
excelente precio. ¡Solicite su cotización!
