# 11 ESTABLECIMIENTO DEL INICIO DE EJECUCIÓN DE PROYECTO

Como se ha indicado anteriormente, el presente proyecto se encuentra construido (desde el año 2007) y

actualmente en operación. Por lo tanto, referido a la cronología de las fases del proyecto, se establecen a

modo referencial hitos de inicio y término para la fase de construcción, dado que sus actividades ya se

encuentran ejecutadas. Misma situación se considera para la fecha e hito de inicio de la fase de operación.

Sin embargo, las fechas e hitos de término de operación son modificadas por el presente proyecto y se

establecen por ejecutar, al igual que para la fase de cierre.

Como se ha señalado, la vida útil de la LTE se relaciona directamente con la vida útil de Planta de

Magnetita. Al respecto, la fase de cierre del proyecto se estima para el año 2041.

En la siguiente tabla se muestra la cronología de las fases del proyecto

|Fecha estimada de término|2007|
