1. OBJETIVO

Entregar pautas que permitan controlar el agua que proviene de las napas subterráneas, debido a la
intersección de ellas, cuando se desarrollan los sondajes durante la etapa de exploraciones y/o
prospecciones.

2. ALCANCE

Este instructivo es válido y aplicable en todos los proyectos de SCMET, tanto al personal propio, como
de empresas colaboradoras.

3. REFERENCIA

Decreto Supremo 132 “Reglamento de Seguridad Minera”.

4. DESCRIPCIÓN

Si producto de la realización de un sondaje se intercepta una napa subterránea, el geólogo debe
evaluar la conveniencia de continuar con el pozo o abandonar dicho pozo. En ambos casos se debe
dar aviso al DGA/MOP a través de una carta.

_4.1 Desarrollo del pozo con presencia de agua_

 - Se procede a sellar las paredes del pozo, de manera que no dañen el pozo así como las
muestras que se obtienen de dicho pozo.

 - Para sellar el pozo se utilizará las mismas barras de perforación que interceptaron la napa,
las que cumplirán la función de cassing.

 - La perforación del pozo continúa con barras de un diámetro menor, las cuales se desplazan
por dentro de las barras que hacen la función de cassing.

 - Si la presión del agua es tal que esta aflora en la superficie (pozo surgente), se procede a
almacenar el agua en piscinas para luego ser utilizada en otras plataformas de sondaje,
regadío de caminos, etc. y se procede a evaluar la continuidad del pozo

_4.2 Abandonar el pozo_

 - Se procede a sacar la columna de barra que se encuentra dentro del pozo.

 - Mientras se retiran las barras, se procede a la confección de un tapo de madera de diámetro
similar al de las barras de perforación utilizadas

 - Una vez confeccionado el tapón, se procede a realizarle hilo en un extremo del tapón y se
acopla a una barra del mismo diámetro.

 - Luego se procede a bajar el tapón con la barra definida para la realización del hilo. El descenso
debe realizarse suavemente por el pozo, y se deja instalado el tapón a la profundidad

deseada.

 - Una vez instalado el tapón, esté producto del agua se hincha produciendo el sellado del pozo.
