**1.10.** **Relación con las políticas y planes evaluados estratégicamente**

El proyecto “Atracadero Ramuntcho” no se relaciones con políticas, planes
y normas que se hayan evaluados estratégicamente

**1.11.** **Aplicabilidad artículo 14, Desarrollo de Proyecto o Actividades por**
**etapas**

El proyecto “Atracadero Ramuntcho” se evaluará desde su etapa inicial
hasta la final, siendo un proyecto único sin fragmentación de éste, así
como tampoco será desarrollado por etapas.
