Declaración de Impacto Ambiental, Proyecto Modificación Lodge y Área de Acampar Paine Grande, Torres del Paine

**3.** **ESTABLECIMIENTO DEL INICIO DE EJECUCIÓN DEL PROYECTO**

La ejecución del Proyecto _Modificación Lodge y Área de Acampar Paine Grande, Torres del Paine_, se dará

inicio una vez obtenida la Resolución de Calificación Ambiental. El inicio de la fase de construcción se

estima para abril del año 2017.

**3.1** **Gestión, Acto o Faena Mínima que da cuenta del inicio de la ejecución del Proyecto**

La acción que establece el inicio de la ejecución del proyecto, es el traslado de materiales para la

construcción de las Plataformas de Domos de Personal.

Fecha Estimada: 30.09.2017

sean entregadas a CONAF.
