**a) Riesgos para la salud de las personas**

**Efectos de una sobreexposición aguda (por una vez):** Irritaciones o quemaduras a la
piel

**Inhalación:** Puede producir mediana a severa irritación si se aspiran los vapores o
aspersiones del producto.

**Contacto con la piel:** Causa severa irritación y quemaduras químicas. El contacto
peligroso puede no causar inmediato dolor

**Contacto con los ojos:** Causa severa irritación o quemadura química. Puede producir
ceguera. Inmediato enjuague es vital en caso de contacto con ojos

**Ingestión:** Puede causar irritación o quemadura química en la garganta, nausea y
malestar estomacal

**Efectos de una sobreexposición crónica (largo plazo):** No se conocen efectos a largo
plazo producidos por el producto
