DIA - Optimización Planta de Sulfuros.

**1.2.9 Establecimiento del inicio de ejecución del Proyecto**

El proyecto comenzará luego de obtenida la RCA favorable, y el hito de inicio lo determina
el comienzo del montaje mecánico asociado a la modificación de circuito de flotación,
correspondiente al inicio de la fase de construcción del proyecto.
