**1.0.- INTRODUCCION**

En el presente documento se detalla el diseño y cálculo de la fundación anular de un estanque
soldado de acido sulfúrico con una capacidad estimada de 432 m3 para uso en caso de
emergencia. Este estanque se ubica dentro de un pretil anular de contención de derrames que
actualmente contiene 2 estanques de acido sulfúrico de similares dimensiones y usos.

También se detalla la verificación del pretil de contención de líquido de derrames por su
modificación en la geometría de trabajo. Actualmente este elemento tiene forma anular y se
modificara su forma a ovalada.

Este nuevo estanque se diseñará de acuerdo a Norma NCh 2369 Of.2003, API 650-2010 y NACE
SP0294-2006.
