# 4 Desarrollo de proyectos o actividades por

etapas

Conforme a la exigencia establecida en el artículo 14 del D.S. No 40/2012 RSEIA, se aclara que el Proyecto **Plan de**
**Manejo de Sedimentos Central Hidroeléctrica La Confluencia**, corresponde a un proyecto ejecutado en una sola
etapa. Por otra parte, no se considera etapa inicial, intermedia o final de un proyecto de mayor envergadura.
