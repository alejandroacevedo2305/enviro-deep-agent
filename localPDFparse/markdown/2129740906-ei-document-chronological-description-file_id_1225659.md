Cronograma del Proyecto

Dado que el Proyecto viene a complementar las actividades de construcción de Línea 6 de
Metro, éste se desarrollará dentro del plazo de las actividades de construcción de la etapa
de construcción de Línea 6. La duración de la etapa de construcción del Proyecto será de
aproximadamente 1.460 días. No obstante lo anterior, la construcción de cada Ventilación
Forzada objeto de la presente DIA durará aproximadamente 180 días (6 meses).

En la Tabla 1, se presentan las actividades y plazos para la construcción de cada
Ventilación Forzada.
