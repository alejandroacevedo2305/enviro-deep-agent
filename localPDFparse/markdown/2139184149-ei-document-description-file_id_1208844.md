Se habilitará dentro de la S/E una zona destinada a las actividades administrativas, acopio de

actividad de retiro de éstas.

final autorizado.

profundas). En lo posible, se emparejara el terreno a pendientes similares a su entorno mediante

Se considera el retiro de las instalaciones provisorias utilizadas para la fase de cierre y limpieza

general del sector, asegurando que el área quede despejada.
