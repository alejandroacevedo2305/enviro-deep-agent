**WGS 84), que muestre la ubicación de las plantas, instalaciones de apoyo,**

**acopios a reprocesar, canchas de acopios, botaderos, etc., descritas en la Tabla**

**N°2 de la DIA. Además se solicita el polígono en coordenadas UTM (Datum WGS**

**84) y la superficie para cada una de las instalaciones y/u obras mencionadas**

**anteriormente**

escala 1:1000 denominados “Emplazamiento General Planta N°1, Fase I, Planta N°2, Fase

General Planta N°3, Fase II”, que muestran la ubicación de las plantas, instalaciones de

apoyo, acopios a reprocesar, canchas de acopios, botaderos, entre otros.

UTM (Datum WGS 84) y la superficie para cada una de las instalaciones y/u obras
