resumen los contenidos de:

1. Descripción del Proyecto (Art. 19, letra a);

efectuar un Estudio de Impacto Ambiental (Art. 19, letra b);

3. Plan de cumplimiento de la legislación ambiental aplicable (Art. 19, letra c);

4. Compromisos ambientales voluntarios (Art. 19, letra d).
