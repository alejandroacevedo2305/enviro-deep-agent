## Declaración de Impacto Ambiental Subestación eléctrica Parinas y línea de transmisión eléctrica Parinas - Taltal

#### Cronograma de las fases del proyecto

De acuerdo con los requerimientos de las letras a.5 y a.6 del artículo 19 del DS N°40/2012 MMA, la DIA
debe incluir un “cronograma de las principales partes, obras y acciones asociadas a esta fase (construcción
y operación), utilizando cualquier herramienta de representación gráfica del progreso del proyecto o

actividad”.

Estimativamente, los plazos de cada fase del proyecto son los siguientes:

### • Construcción y puesta en servicio: comienzo de la construcción será en el segundo semestre de

2021, con una duración de 18 meses.

### • Operación: comienzo de operación comercial se estima ocurrirá en el primer semestre de 2023.

Preliminarmente, no se tiene contemplado el término de esta fase. En caso de requerirse, se

evaluará dentro del segundo semestre de 2053.

### • Cierre: No se prevé el cierre del proyecto. No obstante, se entrega un cronograma ante un eventual

requerimiento de cierre, el cual tendrá una duración preliminar 8 meses para las obras.

A continuación, se entregan los cronogramas para cada una de las fases del proyecto:
