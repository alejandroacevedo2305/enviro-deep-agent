**Métodos de extinción:** Espuma y polvo químico seco. Se puede usar dióxido de

**Medios de extinción** No usar chorro de agua. Debe evitarse el uso de extintores a
**inapropiados:** base de fluorocarbonos ( Halón ) debido a razones ambientales.

**Equipo de protección:** Debe usarse equipo de protección adecuado, incluyendo equipo
de respiración, para aproximarse a un fuego en un espacio
confinado.

usando arena, tierra u otras barreras apropiadas. Si esto no
puede ser prevenido informar a las autoridades locales.
