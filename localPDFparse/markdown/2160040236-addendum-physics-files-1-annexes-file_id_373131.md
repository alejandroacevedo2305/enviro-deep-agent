# Contenedores Metálico 7 m [3]

**Contenedores metálicos.** Avanzados equipos para el manejo de grandes volúmenes

de desechos, preferentemente aquellos con un bajo contenido de humedad; ideal para

residuos industriales y domésticos en grandes cantidades.

**Contenedor cerrado multipropósito** . Versátiles receptáculos utilizados para el manejo

de una amplia variedad de residuos entre los cuales se cuentan los con alta humedad,

los livianos, lodos, los reciclables, etc.

**Descripción**

Características Generales de Contenedores Metálicos

Dimensiones:

7 m3 (202 x 202 x 188 cm) [L x A x H]

Estos contenedores están pensados para disponer en ellos residuos asimilables a

domiciliarios o lo que es lo mismo, basura común.

El uso de contenedores metálicos es óptimo en grandes generadores que disponen de

espacio suficiente de almacenamiento. Debe de elegirse un lugar específico para ello y

cuyo acceso no sea interrumpido para que el camión que realiza el servicio pueda

acceder a los contenedores para el retiro de los residuos.
