# ANEXO 6. INFORME DE PERMEABILIDAD DEL TERRENO

**PRUEBA DE INFILTRACION PARA DISEÑO DE ZANJA DE INFILTRACION**

La zanja de infiltración o campo de riego, es el elemento utilizado para descargar los
líquidos provenientes de las plantas de tratamiento, permitiendo su oxidación y
disposición final.

El procedimiento para realizar la prueba es la siguiente:

1.- Se realizaron 3 calicatas de prueba, esparcidos uniformemente dentro del área
propuesta para el campo de infiltración. Las excavaciones para realizar las pruebas
deben tener 0.30 m por lado y 0.60 m de profundidad.

2.- Se raspó cuidadosamente el fondo y las paredes del agujero perforado, para
remover cualquier superficie de suelo compactado y que pueda filtrarse el agua. Se
colocaron 5 centímetros de arena gruesa o gravilla para proteger el fondo contra
socavaciones y sedimentos.

3.- Saturación del suelo

Realizar excavación de dimensiones 0,30 x 0,30 mts de base y 0,20 de profundidad.

Llénese con agua ésta excavación y deje que ésta se filtre totalmente.

Colóquese agua nuevamente, hasta una altura de 0,15cms. y obsérvese el tiempo
requerido para que el nivel del agua baje 2 1/2 cms.

**INDICE DE PERMEABILIDAD DEL SUELO (Ip)**

Está basado en el grafico denominado “ _**Ensayo para determinar Índice de**_
_**absorción**_ ” del Reglamento de Alcantarillados Particulares:

Ip = Índice de permeabilidad (Litros/M [2] /día)
Td = Tiempo más desfavorable (Minutos)

Ip = 237,36 x Td -0,578

Ip = 237,36 x 5,46 -0,578

Ip = 88,98 (Litros/M [2] /día)
