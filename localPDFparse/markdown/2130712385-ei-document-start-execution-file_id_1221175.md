2 Establecimiento del inicio de ejecución del

Proyecto

El inicio de la ejecución sistemática y permanente del Proyecto **Parque Solar Fotovoltaico Luz del Oro**
estará dado por la instalación de faenas principal y la habilitación del camino de acceso. Esta tarea se
proyecta para el primer trimestre 2017. Lo anterior, conforme a la indicación establecida en el artículo 16

del D.S. No 40/2013 RSEIA.
