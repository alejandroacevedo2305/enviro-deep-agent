**4.** **PLANIFICACIÓN DEL MUESTREO**

La planificación del muestreo se basó en la recopilación de antecedentes a través de consulta telefónica y por correo
electrónico, recepción de fotografías, entre otros. No hubo visita previa.

El plan de muestreo describe el diseño de muestreo y el procedimiento a seguir para obtener una muestra
representativa del residuo a caracterizar. La selección de los procedimientos de muestreo se basa en las siguientes
normas internacionales:

La Norma ASTM D4687, “Standard guide for general planning of waste sampling”, es una guía que permite establecer, la
selección de los puntos de muestreo, la forma de ejecutarlo, el número apropiado de muestras y el equipamiento
necesario. Los esquemas más comunes para la selección de los puntos de muestreo son:

Muestreo directo o de juicio: Sistema ampliamente
utilizado cuando se tiene alguna restricción de recurso.
Este es aplicable cuando se conoce muy bien la muestra
o para establecer criterios a partir del residuo.

Sistema de coordenadas: Las muestras son tomadas
aleatoriamente desde un punto preestablecido hasta
intervalos regulares a lo largo de la coordenada. Este
sistema se utiliza cuando se requiere que la población
tenga igual probabilidad de ser muestreada; se pueden
utilizar tablas de números al azar.
