Estas serán realizadas cada 3 meses el primer año de<br>la reforestación, en el segundo y tercer año se realizarán cada 6 meses. Cabe<br>señalar que la medida se dará por cumplida una vez que se acrdite el<br>prendimiento del 75% de los individuos reforestados, y que el monitoreo<br>continuará hasta que se pueda demostrar ese prendimiento.
