**Plan de cuidados durante la construcción de la línea de flujo**

Previo al inicio de faenas, se realizará una inducción a todo el personal involucrado en
las labores de construcción del ducto, impartida por un profesional del área (biólogo o
veterinario).

Antes de comenzar la construcción del ducto, se realizará una inspección del área con
el objeto de determinar si las madrigueras aún se encuentran inactivas o han vuelto a

ser utilizadas.

El sector de madrigueras será delimitado para impedir el ingreso de personas o
vehículos, incluyendo señalética específica para este fin.

**operación**
